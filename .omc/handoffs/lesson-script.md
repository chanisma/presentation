## 슬라이드 4: class와 id 개념 설명 (class와 id란?)

자, 먼저 class와 id에 대해 알아볼게요. 지난 시간에 CSS로 `p` 태그나 `h1` 태그 전체에 스타일을 줬잖아요? 그런데 만약 `p` 태그가 10개인데, 그중 딱 3개만 빨간색으로 만들고 싶으면 어떻게 할까요? 바로 class와 id를 사용하면 돼요!

class는 여러 요소에 같은 이름을 붙일 수 있어요. 쉽게 말해서 "학년"이랑 비슷해요. 1학년이 여러 명이듯, 같은 class를 가진 요소도 여러 개일 수 있어요. CSS에서는 점(`.`)을 찍어서 `.클래스이름`으로 선택해요.

id는 하나의 요소에만 붙이는 고유한 이름이에요. "출석번호"처럼 딱 하나만 있어야 해요. CSS에서는 샵(`#`)을 붙여서 `#아이디이름`으로 선택해요.

## 슬라이드 5: class와 id 실습 (식별자에 CSS 설정하기 실습)

자, 이제 직접 해볼게요! 왼쪽 코드를 보면 전체 HTML 문서가 보이죠? 위쪽에 `<style>` 태그 안에 CSS가 있고, 아래쪽 `<body>` 안에 HTML 요소들이 있어요. 이 전체 구조를 눈으로 확인해보세요.

지금 `.box`라는 class가 모든 박스에 적용되어 있고, `.highlight`는 두 번째 박스에만, `#special`은 세 번째 박스에만 적용되어 있어요. 오른쪽 결과를 보면 각각 다르게 보이죠?

**여러분이 해볼 것**:
1. `.highlight`의 `color: red;`를 `color: blue;`로 바꿔보세요. 어떤 박스가 변하나요?
2. `<style>` 안에 `.big { font-size: 24px; }` 를 새로 추가해보세요. 그리고 아래 body에서 원하는 박스의 class에 `big`을 추가해보세요.
3. `#special`의 `background-color`를 `pink`나 다른 색으로 바꿔보세요.

## 슬라이드 6: 자손 선택자와 자식 선택자 개념 설명

다음은 선택자를 좀 더 정교하게 사용하는 방법이에요. HTML 요소들은 부모-자식 관계를 가지고 있잖아요? `<ul>` 안에 `<li>`가 있고, 그 `<li>` 안에 또 `<ul>`이 있을 수 있어요.

**자손 선택자**는 띄어쓰기로 표현해요. `A B`라고 쓰면 A 안에 있는 모든 B를 선택해요. 손자, 증손자까지 전부 다요!

**자식 선택자**는 `>`로 표현해요. `A > B`라고 쓰면 A 바로 아래에 있는 B만 선택해요. 직계 자식만!

쉽게 생각하면, 자손 선택자는 "우리 가족 중 모든 아이들"이고, 자식 선택자는 "내 자녀만"이에요.

## 슬라이드 7: 자손/자식 선택자 실습 (선택자 응용 실습)

이번 코드에는 메뉴 목록이 있어요. `<ul class="menu">` 안에 `<li>`가 있고, 그 안에 또 `<ul>`과 `<li>` 서브메뉴가 있죠. 전체 HTML 구조를 보면서, 어떤 요소가 어떤 요소 안에 들어있는지 확인해보세요.

**여러분이 해볼 것**:
1. `.menu li`의 `color`를 `green`으로 바꿔보세요. 메뉴 전체가 초록색으로 변하는 게 보이죠? 자손 선택자니까 안에 있는 모든 `li`가 다 바뀌어요.
2. `.menu > li`에 `color: red;`를 추가해보세요. 이번엔 1단계 메뉴만 빨간색으로 변할 거예요. 자식 선택자니까요!
3. 서브메뉴만 보라색으로 만들어볼까요? `.menu ul li { color: purple; }`를 추가해보세요.

## 슬라이드 8: Margin과 Padding 개념 설명 (박스 모델의 핵심)

자, 이제 여백에 대해 배워볼게요. CSS에서 모든 요소는 "박스"예요. 이 박스는 안쪽부터 content(내용) → padding(안쪽 여백) → border(테두리) → margin(바깥 여백) 순서로 구성되어 있어요.

**padding**은 박스 안쪽 여백이에요. 액자로 비유하면, 그림과 액자 틀 사이의 여백이에요. 내용물이 테두리에 바짝 붙지 않도록 숨통을 틔워주는 거죠.

**margin**은 박스 바깥쪽 여백이에요. 액자와 액자 사이의 간격이라고 생각하면 돼요. 요소들이 서로 붙지 않도록 거리를 두는 거예요.

방향도 지정할 수 있어요. `margin-top`, `margin-right`, `margin-bottom`, `margin-left` 이렇게요. `margin: 10px 20px;`처럼 쓰면 위아래 10px, 좌우 20px이에요.

이번 코드에는 세 개의 박스가 있어요. 각각 다른 여백 설정이 되어 있으니 결과를 잘 비교해보세요.

## 슬라이드 10: Flexbox 개념 설명 (가로 세로 배치의 마법사)

마지막으로 Flexbox를 배워볼게요! 지금까지 요소들이 기본적으로 위에서 아래로 쌓였잖아요? 그런데 프로필 사진 옆에 소개글을 나란히 놓고 싶으면 어떻게 할까요? 바로 Flexbox를 사용하면 돼요!

사용법은 간단해요. 부모 요소에 `display: flex;`만 붙이면 자식들이 가로로 정렬돼요.

그 다음에 세부 조정을 할 수 있어요:
- **flex-direction**: `row`면 가로, `column`이면 세로 배치
- **justify-content**: 주축 방향 정렬. `center`면 가운데, `space-between`이면 양끝에 붙고 나머지 균등 배치
- **align-items**: 교차축 방향 정렬. `center`면 세로 가운데
- **gap**: 항목 사이 간격

## 슬라이드 11: Flexbox 배치 실습

이번 코드에는 네 개의 아이템이 flex 컨테이너 안에 들어있어요. 지금은 `flex-direction: row;`라서 가로로 나란히 배치되어 있죠?

**여러분이 해볼 것**:
1. `flex-direction`을 `column`으로 바꿔보세요. 세로로 쌓이는 게 보이죠?
2. 다시 `row`로 돌려놓고, `justify-content`를 `center`로 바꿔보세요. 가운데로 모이죠?
3. `justify-content`를 `space-between`으로 바꿔보세요. 양쪽 끝에 붙으면서 균등하게 분배될 거예요.
4. `align-items`를 `flex-end`로 바꿔보세요. 아이템들이 아래쪽으로 내려갈 거예요.

이것저것 조합해서 바꿔보세요! Flexbox는 직접 만져봐야 감이 와요.
