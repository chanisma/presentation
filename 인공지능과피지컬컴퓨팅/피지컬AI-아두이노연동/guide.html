손동작 인식 스마트 조명 — 학생용 실습 가이드

0 프로젝트 개요

웹캠으로 손동작을 인식하고, 아두이노에 연결된 LED를 제어하는 피지컬 AI 프로젝트입니다.

전체 시스템 구조

웹캠 → Teachable Machine → Web Serial API → USB → 아두이노 → LED (센서) (AI 모델) (통신) (케이블) (제어기) (출력)

필요한 파일

파일용도설명
app.html웹 앱Chrome에서 열어 사용하는 메인 프로그램
smart_light.ino아두이노 코드Arduino IDE로 아두이노에 업로드
guide.html이 문서실습 가이드 (지금 보고 있는 문서)

준비물 체크리스트

  • 아두이노 Uno + USB 케이블
  • LED 3개 (빨강, 초록, 파랑)
  • 220Ω 저항 3개
  • 브레드보드 + 점퍼선
  • Chrome 브라우저 (Edge도 가능)
  • Arduino IDE 설치됨
  • Teachable Machine 모델 링크 (지난 시간에 저장한 것)

1 아두이노 회로 연결

회로 연결표

LED 색상아두이노 핀연결 방법
빨간 LED디지털 9번긴 다리(+) → 220Ω 저항 → 9번 핀
초록 LED디지털 10번긴 다리(+) → 220Ω 저항 → 10번 핀
파란 LED디지털 11번긴 다리(+) → 220Ω 저항 → 11번 핀
모든 LED의 짧은 다리(-) → GND에 연결

회로도

아두이노 Uno ┌──────────────┐ │ │ │ 9번 핀 ──── 220Ω ──── [빨강 LED] ──┐ │ │ │ │ 10번 핀 ──── 220Ω ──── [초록 LED] ──┤ │ │ │ │ 11번 핀 ──── 220Ω ──── [파랑 LED] ──┤ │ │ │ │ GND ────────────────────────────────┘ │ │ └──────────────┘

주의! LED에는 극성이 있습니다. 긴 다리가 (+), 짧은 다리가 (-) 입니다. 반대로 연결하면 켜지지 않습니다. 저항을 빼먹으면 LED가 타버릴 수 있으니 반드시 연결하세요.

2 아두이노 코드 업로드

  1. 1

    Arduino IDE 실행

    아두이노를 USB 케이블로 컴퓨터에 연결한 후 Arduino IDE를 엽니다.

  2. 2

    보드 & 포트 설정

    메뉴에서 도구 → 보드 → "Arduino Uno" 선택, 도구 → 포트 → "COM○" (숫자가 있는 것) 선택

  3. 3

    코드 붙여넣기

    Arduino IDE의 기존 코드를 모두 지우고, 아래 코드를 복사하여 붙여넣으세요. 또는 smart_light.ino 파일을 직접 열어도 됩니다.

smart_light.ino
const int RED_PIN = 9; const int GREEN_PIN = 10; const int BLUE_PIN = 11; void setup() { Serial.begin(9600); pinMode(RED_PIN, OUTPUT); pinMode(GREEN_PIN, OUTPUT); pinMode(BLUE_PIN, OUTPUT); // 시작 시 LED 3번 깜빡 → 연결 확인용 for (int i = 0; i < 3; i++) { digitalWrite(RED_PIN, HIGH); digitalWrite(GREEN_PIN, HIGH); digitalWrite(BLUE_PIN, HIGH); delay(200); digitalWrite(RED_PIN, LOW); digitalWrite(GREEN_PIN, LOW); digitalWrite(BLUE_PIN, LOW); delay(200); } } void loop() { if (Serial.available() > 0) { char cmd = Serial.read(); digitalWrite(RED_PIN, LOW); digitalWrite(GREEN_PIN, LOW); digitalWrite(BLUE_PIN, LOW); switch (cmd) { case 'R': digitalWrite(RED_PIN, HIGH); break; case 'G': digitalWrite(GREEN_PIN, HIGH); break; case 'B': digitalWrite(BLUE_PIN, HIGH); break; case 'O': break; // 이미 전부 꺼짐 } } }
  1. 4

    업로드

    상단의 → (업로드) 버튼을 클릭합니다. "Done uploading." 메시지가 나오면 성공!

  2. 5

    확인

    업로드 성공 시 LED 3개가 3번 깜빡입니다. 깜빡이면 회로와 코드 모두 정상!

중요! 업로드가 끝나면 Arduino IDE의 시리얼 모니터를 반드시 닫아주세요. 시리얼 모니터가 열려 있으면 웹 페이지에서 아두이노에 연결할 수 없습니다.

3 웹 앱(app.html) 사용법

  1. 1

    app.html을 Chrome에서 열기

    파일을 더블클릭하거나, Chrome에 파일을 드래그합니다.

  2. 2

    모델 URL 입력 → "모델 로드" 클릭

    지난 시간에 Teachable Machine에서 저장한 모델 링크를 붙여넣고 "모델 로드" 버튼을 클릭합니다. 웹캠이 자동으로 켜집니다.

  3. 3

    "아두이노 연결" 클릭

    팝업에서 아두이노 포트를 선택합니다. (보통 "USB Serial" 또는 "COM○"으로 표시됩니다)

  4. 4

    손동작으로 LED 제어!

    웹캠 앞에서 손동작을 하면 AI가 인식하고, 아두이노 LED가 제어됩니다.

클래스 → LED 매핑

TM 클래스 순서전송 명령LED 동작
1번째 클래스 (예: 주먹)R빨간 LED 켜짐
2번째 클래스 (예: 보)G초록 LED 켜짐
3번째 클래스 (예: 가위)B파란 LED 켜짐
4번째 클래스 (예: 없음)O모든 LED 꺼짐

팁: AI의 인식 확률이 70% 이상일 때만 명령이 전송됩니다. 동작을 명확하게 해주세요. 화면 왼쪽의 LED 표시등으로 현재 상태를 확인할 수 있습니다.

4 문제 해결 가이드

LED가 안 켜져요

1. LED 극성 확인 (긴 다리 = +)

2. 저항이 연결되어 있는지 확인

3. 핀 번호가 9, 10, 11인지 확인

4. 아두이노에 코드가 업로드 되었는지 확인

시리얼 연결이 안 돼요

1. Arduino IDE 시리얼 모니터 닫기

2. Chrome 브라우저인지 확인

3. USB 케이블 다시 연결

4. 페이지 새로고침 후 재시도

모델 로드가 안 돼요

1. URL 끝에 / 가 있는지 확인

2. 인터넷 연결 확인

3. URL을 다시 복사해서 붙여넣기

4. Teachable Machine에서 모델 재업로드

인식이 잘 안 돼요

1. 밝은 곳에서 테스트

2. 동작을 더 크고 명확하게

3. 배경이 단순한 곳에서 테스트

4. TM에서 데이터 추가 후 재학습

포트가 목록에 안 보여요? Windows에서 아두이노 드라이버가 설치되지 않았을 수 있습니다. Arduino IDE를 설치하면 드라이버가 함께 설치됩니다. 설치 후 USB를 다시 연결하세요.

5 코드 이해하기 (선택)

아두이노 코드가 하는 일

코드의미
Serial.begin(9600)시리얼 통신 시작 (속도 9600bps)
pinMode(9, OUTPUT)9번 핀을 출력 모드로 설정
Serial.available()시리얼로 데이터가 왔는지 확인
Serial.read()시리얼에서 문자 1개 읽기
digitalWrite(9, HIGH)9번 핀에 전류 보내기 (LED 켜기)

웹 앱(app.html)이 하는 일

단계설명
1. 모델 로드TensorFlow.js로 Teachable Machine 모델을 불러옴
2. 웹캠 캡처웹캠 영상을 프레임 단위로 캡처
3. 예측캡처한 이미지를 AI 모델에 넣어 각 클래스의 확률 계산
4. 판단가장 높은 확률의 클래스를 선택 (70% 이상일 때만)
5. 전송Web Serial API로 해당 문자(R/G/B/O)를 아두이노에 전송

+ 도전 과제

기본 프로젝트를 완성했다면, 아래 도전 과제에 도전해보세요!

  1. A

    동작 클래스 추가하기

    서보모터 연결하기

    서보모터를 연결하면 손동작으로 물리적인 움직임(문 열기, 팔 움직이기 등)을 제어할 수 있습니다.
