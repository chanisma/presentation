⚡ Step 1 — 아두이노 회로 구성

LED 3개를 아두이노에 연결합니다

  1. 01

    빨간 LED → 디지털 9번 핀

    LED 긴 다리(+) → 220Ω 저항 → 9번 핀, 짧은 다리(-) → GND

  2. 02

    초록 LED → 디지털 10번 핀

    같은 방식으로 10번 핀에 연결

  3. 03

    파란 LED → 디지털 11번 핀

    같은 방식으로 11번 핀에 연결

LED 연결 규칙

  • LED는 극성이 있습니다 — 긴 다리가 (+), 짧은 다리가 (-)
  • 저항을 빼먹으면 LED가 타버릴 수 있으니 반드시 연결하세요

💻 Step 2 — 아두이노 코드 업로드

// LED 핀 번호 설정 const int RED = 9; const int GREEN = 10; const int BLUE = 11; void setup() { Serial.begin(9600); // 시리얼 통신 시작 pinMode(RED, OUTPUT); // LED 핀을 출력 모드로 pinMode(GREEN, OUTPUT); pinMode(BLUE, OUTPUT); } void loop() { if (Serial.available()) { // 데이터가 들어오면 char cmd = Serial.read(); // 문자 1개 읽기 // 모든 LED 끄기 digitalWrite(RED, LOW); digitalWrite(GREEN, LOW); digitalWrite(BLUE, LOW); // 받은 명령에 따라 LED 켜기 if (cmd == 'R') digitalWrite(RED, HIGH); // 주먹 → 빨강 if (cmd == 'G') digitalWrite(GREEN, HIGH); // 보 → 초록 if (cmd == 'B') digitalWrite(BLUE, HIGH); // 가위 → 파랑 // 'O'이면 전부 꺼진 상태 유지 } }

시리얼로 문자 하나를 받아 LED를 제어합니다. 'R'=빨강, 'G'=초록, 'B'=파랑, 'O'=전부 끄기

📝 웹 페이지 코드 핵심

TM 모델 로드 + 시리얼 전송 (코드 파일은 제공됩니다)

// 1. TM 모델 로드 const model = await tmImage.load(modelURL, metadataURL); // 2. 웹캠으로 예측 const prediction = await model.predict(webcam.canvas); // 3. 가장 높은 확률의 클래스 찾기 let bestClass = prediction.reduce( (a, b) => a.probability > b.probability ? a : b ); // 4. 시리얼로 명령 전송 if (bestClass.className === "주먹") writer.write("R"); // 빨간 LED

⚙ 아두이노 업로드 가이드

  1. 01

    Arduino IDE 실행

    아두이노를 USB로 연결한 후 Arduino IDE를 엽니다

  2. 02

    보드 & 포트 선택

    도구 → 보드: "Arduino Uno" 선택, 포트: "COM○" (숫자가 붙은 것) 선택

  3. 03

    코드 붙여넣기 → 업로드

    기존 코드를 모두 지우고, 제공된 코드를 붙여넣은 후 업로드(→) 버튼 클릭

주의

  • 업로드 시 "Done uploading" 메시지가 나오면 성공
  • 포트가 안 보이면 USB 케이블을 다시 연결하거나 드라이버 설치 필요
  • 업로드 후에는 Arduino IDE의 시리얼 모니터를 반드시 닫아주세요
