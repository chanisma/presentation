CNN이란?
🧠

CNN (Convolutional Neural Network)

합성곱 신경망 - 이미지 인식에 특화된 인공지능

🔤 이름 풀어보기

Convolutional = 합성곱 (필터로 특징 추출)
Neural Network = 신경망 (뇌의 뉴런을 모방)

💡 비유로 이해하기

CNN은 마치 탐정과 같습니다!
① 사진에서 단서(특징)를 찾고
② 단서들을 조합하여
③ 최종 결론(분류)을 내립니다

3

💡 CNN Explainer의 Tiny VGG 모델

CNN Explainer에서 사용하는 Tiny VGG는 2개의 Conv-ReLU-Pool 블록과 1개의 완전연결층으로 구성된 간단한 CNN 모델입니다. 이 구조만으로도 에스프레소, 식료품점 등 10가지 클래스를 분류할 수 있습니다!

Step 1

Step 2: 컨볼루션 (Convolution)
Step 2 · 핵심 연산

컨볼루션 - 특징을 추출하는 필터

💡 비유

돋보기로 사진의 작은 영역을 하나씩 살펴보며
"여기에 가장자리가 있네!"
"여기에 무늬가 있네!"
라고 특징을 기록하는 것과 같습니다.

🔍 작동 원리

작은 필터(커널)가 이미지 위를 슬라이딩하며
각 위치에서 요소별 곱셈 → 합산을 수행합니다.
이 결과가 특징 맵(Feature Map)이 됩니다.

🎯 필터의 종류

• 수평 엣지 필터: 가로 방향 경계선 감지
• 수직 엣지 필터: 세로 방향 경계선 감지
• 샤프닝 필터: 윤곽을 선명하게
※ CNN은 학습을 통해 최적의 필터를 스스로 찾아냅니다!

특징 맵 (Feature Maps)
핵심 개념

필터가 만들어내는 특징 맵

하나의 컨볼루션 층에는 여러 개의 필터가 있습니다. 각 필터는 서로 다른 특징을 감지하여 각각의 특징 맵을 만듭니다.

↔️

필터 1: 수평 엣지

가로 방향의 경계선을 감지합니다.
예: 지평선, 선반 가장자리

↕️

필터 2: 수직 엣지

세로 방향의 경계선을 감지합니다.
예: 기둥, 문 가장자리

↗️

필터 3: 대각선 엣지

대각선 방향의 특징을 감지합니다.
예: 지붕 경사, 대각선 패턴

🔑 CNN Explainer에서 Tiny VGG의 첫 번째 층에는 10개의 필터가 있어
10개의 서로 다른 특징 맵을 생성합니다. 각 특징 맵은 이미지의 다른 측면을 포착합니다!
💡 비유

10명의 탐정이 같은 사진을 보지만, 각자 다른 관점에서 단서를 찾는 것과 같습니다.
한 명은 윤곽선을, 다른 한 명은 색상 변화를, 또 다른 한 명은 질감을 집중해서 봅니다.

Step 3: ReLU 활성화 함수
Step 3

ReLU - 음수를 없애는 활성화 함수

📌 ReLU란? (Rectified Linear Unit)

놀라울 정도로 단순한 함수입니다:

양수 → 그대로 유지
음수 → 0으로 변환

ReLU(x) = max(0, x)

📈 ReLU 함수 그래프

💡 비유

탐정이 찾은 단서 중에서
"확실한 단서만 남기고 (양수 = 특징 있음)
애매한 건 무시하는 것" (음수 = 특징 없음)
과 같습니다.

🧮 ReLU 계산 예시

입력 (x) 3.2 -1.5 0.7 -4.1 2.0 -0.3
max(0, x) 3.2 0 0.7 0 2.0 0

❓ 왜 ReLU가 필요할까?

비선형성(Nonlinearity)을 추가하기 위해서입니다!

ReLU 없이 컨볼루션만 반복하면, 아무리 층을 쌓아도 단순한 직선적 변환에 불과합니다. ReLU가 있어야 복잡한 패턴을 학습할 수 있습니다.

Step 4: 맥스 풀링 (Max Pooling)
Step 4

맥스 풀링 - 중요한 특징만 남기기

📉 맥스 풀링이란?

특징 맵을 작은 영역(2×2)으로 나누고,
각 영역에서 가장 큰 값(최댓값)만 선택합니다.

결과: 크기가 절반으로 줄어듭니다!

💡 비유

시험 성적표에서 각 과목군의 최고 점수만 뽑아 요약하는 것과 같습니다. 세부 사항은 사라지지만, 가장 중요한 정보는 남습니다.

✅ 맥스 풀링의 장점

① 계산량 감소: 데이터 크기가 1/4로
② 과적합 방지: 불필요한 세부 사항 제거
③ 위치 불변성: 특징의 정확한 위치보다
   "특징이 있는지"가 더 중요

Conv → ReLU → Pool 반복의 의미

CNN에서 같은 블록을 반복하면, 층이 깊어질수록 더 복잡하고 추상적인 특징을 학습합니다.

🔵

1번째 블록

저수준 특징

단순한 패턴을 감지합니다:
• 가장자리 (엣지)
• 색상의 변화
• 밝기 차이
• 간단한 질감

🟣

2번째 블록

중수준 특징

패턴의 조합을 감지합니다:
• 모서리, 꼭짓점
• 곡선, 원형
• 반복되는 질감
• 간단한 도형

🟠

더 깊은 층 (심화 CNN)

고수준 특징

의미 있는 부분을 감지합니다:
• 눈, 코, 입
• 바퀴, 창문
• 꽃잎 모양
• 글자 형태

🔑 핵심 포인트: CNN은 층이 깊어질수록 "선 → 모양 → 물체 부분 → 전체 물체"를 순서대로 인식합니다.
이것이 CNN이 이미지를 "이해"하는 방식입니다!
💡 CNN Explainer에서 확인!

CNN Explainer에서 첫 번째 Conv 층의 특징 맵은 단순한 패턴(엣지, 색상)을 보여주고,
두 번째 Conv 층의 특징 맵은 더 복잡한 패턴(질감, 모양 조합)을 보여줍니다.

Step 5~6

🧮 완전연결층 (Dense Layer)

모든 입력 뉴런이 모든 출력 뉴런에 연결됩니다.

추출된 특징들을 종합적으로 분석하여
"이 특징 조합은 고양이일 가능성이 높다"는

CNN의 학습: 역전파
🔄

CNN은 어떻게 학습할까?

역전파(Backpropagation) - 실수에서 배우는 과정

1

순전파 (Forward Pass)

이미지를 입력 → CNN 파이프라인을 거쳐 → 예측 결과 출력
예: "이 이미지는 강아지(70%)"라고 예측

2

손실 계산 (Loss)

예측이 정답과 얼마나 틀렸는지 계산합니다
정답은 "고양이"인데 "강아지"라고 했으므로 → 큰 손실!

3

역전파 (Backward Pass)

출력 → 입력 방향으로 오차의 원인을 추적합니다
"어떤 필터의 어떤 가중치가 잘못된 결과를 만들었는가?"

4

가중치 업데이트

오차를 줄이는 방향으로 필터의 숫자(가중치)를 조정합니다
다음에는 고양이를 더 잘 맞출 수 있도록 필터 수정!

🔁 이 과정을 수천~수만 번 반복하면, CNN의 필터들은 점점 더 정확한 특징을 잡아내도록 발전합니다.
사람이 필터를 설계하는 것이 아니라, CNN이 스스로 최적의 필터를 학습합니다!

🔍 컨볼루션 실제 처리 결과

샘플 이미지에 3×3 엣지 감지 필터를 적용한 결과입니다. 필터가 이미지의 가장자리(엣지)를 찾아냅니다.

입력 이미지
→
수평 엣지 필터 결과
수직 엣지 필터 결과
💡 밝은 부분은 필터가 강하게 반응한 곳(특징 발견), 어두운 부분은 반응이 없는 곳입니다. 서로 다른 필터가 서로 다른 방향의 가장자리를 감지하는 것을 확인하세요!

🗺️ 여러 필터의 특징 맵

같은 이미지에 6개의 서로 다른 필터를 적용하면, 각각 다른 특징을 감지하는 6개의 특징 맵이 만들어집니다.

💡 각 특징 맵은 이미지의 서로 다른 측면을 포착합니다. CNN은 이 여러 관점의 정보를 종합하여 이미지를 이해합니다.

⚡ ReLU 적용 전후 비교

컨볼루션 결과에 ReLU를 적용하면 음수 값이 모두 0(검정)으로 변합니다.

컨볼루션 결과 (ReLU 전)
음수 = 빨간 계열
→
ReLU
ReLU 적용 후
양수만 남음, 음수→0(검정)
💡 빨간 영역(음수)이 모두 검정(0)으로 바뀐 것을 확인하세요. 이렇게 "확실한 특징"만 남기는 것이 ReLU의 역할입니다.

📉 맥스 풀링 실제 처리 결과

2×2 맥스 풀링을 적용하면 이미지 크기가 절반으로 줄어들지만, 중요한 특징은 보존됩니다.

풀링 전 (128×128)
→
2×2 max
풀링 후 (64×64)
💡 크기가 절반으로 줄었지만, 가장자리와 주요 특징은 그대로 남아있습니다! 이렇게 계산량을 줄이면서도 핵심 정보를 유지하는 것이 풀링의 장점입니다.
