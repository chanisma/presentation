엑추에이터 소개

⊕ 엑추에이터란?

전기 신호를 물리적 동작으로 바꾸는 출력 장치

피에조 부저

전기 신호 → 소리 출력. tone() 함수로 주파수 조절 가능. 경고음, 멜로디 등에 활용

난이도 ★☆☆

LED (발광 다이오드)

전기 신호 → 빛 출력. analogWrite()로 밝기 조절(PWM). 지난 시간에 사용한 부품

난이도 ★☆☆

SG90 서보모터

전기 신호 → 회전 동작. 0°~180° 각도 제어. 문 개폐, 팔 움직임 등 동작 표현

난이도 ★★☆

이 외에도 DC 모터, 진동 모터, 릴레이 등 다양한 엑추에이터가 있지만, 오늘은 위 세 가지를 활용합니다.

프롬프트 예시 — 부저

✦ 프롬프트 예시: 피에조 부저 제어

AI에게 보낼 프롬프트 아두이노 프로그래밍 전문가로서, 아래 코드를 기반으로 수정해 주세요.

[기존 코드 구조]
시리얼로 문자를 수신하면 switch문으로 분기하는 구조입니다.

[요청 사항]
✦ 엑추에이터: 피에조 부저 (8번 핀 연결)
✦ 동작 조건:
  - 'A' 수신 → 도레미 멜로디 재생
  - 'B' 수신 → 경고음(높은 주파수 반복)
  - 'C' 수신 → 소리 정지
✦ tone() 함수를 사용해 주세요.
// 기본 코드 틀 (이 코드를 프롬프트에 함께 붙여넣기) const int BUZZER_PIN = 8; void setup() { Serial.begin(9600); pinMode(BUZZER_PIN, OUTPUT); } void loop() { if (Serial.available() > 0) { char cmd = Serial.read(); switch (cmd) { case 'A': /* 멜로디 재생 */ break; case 'B': /* 경고음 */ break; case 'C': noTone(BUZZER_PIN); break; } } }
프롬프트 예시 — 서보모터

◆ 프롬프트 예시: SG90 서보모터 제어

AI에게 보낼 프롬프트 아두이노 전문가로서, 아래 코드를 기반으로 수정해 주세요.

[요청 사항]
✦ 엑추에이터: SG90 서보모터 (9번 핀 연결)
✦ Servo 라이브러리를 사용해 주세요.
✦ 동작 조건:
  - 'L' 수신 → 서보 0° (왼쪽)
  - 'C' 수신 → 서보 90° (중앙)
  - 'R' 수신 → 서보 180° (오른쪽)
✦ 각도 이동은 부드럽게(10ms 딜레이로 점진 이동)
// 기본 코드 틀 (이 코드를 프롬프트에 함께 붙여넣기) #include <Servo.h> Servo myServo; const int SERVO_PIN = 9; int currentAngle = 90; void setup() { Serial.begin(9600); myServo.attach(SERVO_PIN); myServo.write(90); // 중앙 위치로 초기화 } void loop() { if (Serial.available() > 0) { char cmd = Serial.read(); switch (cmd) { case 'L': /* 0도로 이동 */ break; case 'C': /* 90도로 이동 */ break; case 'R': /* 180도로 이동 */ break; } } }

✦ 실습 진행 순서

단계별로 따라 하면 완성됩니다

  1. 01

    프로젝트 구상

    웹캠으로 인식할 대상(손동작, 표정, 사물 등)과 사용할 엑추에이터를 결정합니다.

  2. 02

    Teachable Machine 모델 준비

    지난 시간에 만든 모델을 재사용하거나, 새 모델을 학습시킵니다. 클래스를 3개 이상 만드세요.

  3. 03

    AI로 아두이노 코드 생성

    앞서 배운 프롬프트 예시를 참고하여 AI에게 코드를 요청하고, 생성된 코드를 아두이노에 업로드합니다.

  4. 04

    회로 연결 + 테스트

    엑추에이터를 아두이노에 연결하고, 웹 앱에서 시리얼 연결 후 동작을 확인합니다.
