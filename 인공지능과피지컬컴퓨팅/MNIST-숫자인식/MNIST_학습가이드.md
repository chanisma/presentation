# CNN 기반 MNIST 손글씨 숫자 인식 학습 가이드

본 학습 가이드는 합성곱 신경망(CNN)을 활용하여 0부터 9까지의 손글씨 숫자를 인식하는 인공지능 모델 생성 과정을 학습하기 위해 제작되었습니다. 인공지능과 피지컬 컴퓨팅 수업의 실습 자료를 바탕으로 핵심 개념을 정리하고 학습 내용을 점검할 수 있도록 구성되었습니다.

---

## 1. 주요 학습 내용 요약

| 구분 | 주요 내용 |
| :--- | :--- |
| **학습 목표** | 합성곱 신경망(CNN)의 구조를 이해하고, MNIST 데이터셋을 활용해 손글씨 숫자를 분류하는 AI 모델을 직접 구현한다. |
| **대상 데이터** | **MNIST 데이터셋**: 0부터 9까지의 숫자로 이루어진 손글씨 이미지 데이터. |
| **핵심 기술** | **합성곱 신경망 (CNN)**: 이미지의 특징을 추출하고 학습하는 데 최적화된 신경망 구조. |
| **실습 환경** | 인공지능 모델 구현 및 피지컬 컴퓨팅 연계 학습. |

---

## 2. 단답형 연습 문제

**문항 1.** 이번 실습에서 사용하는 데이터셋의 명칭으로, 0부터 9까지의 손글씨 숫자로 구성된 데이터의 이름은 무엇입니까?
> **정답:** MNIST

**문항 2.** 이미지 인식 및 분류에 특화된 신경망으로, 이번 실습에서 인공지능 모델을 만들기 위해 사용하는 알고리즘의 명칭은 무엇입니까?
> **정답:** 합성곱 신경망 (또는 CNN)

**문항 3.** MNIST 데이터셋을 통해 학습된 인공지능 모델이 인식할 수 있는 숫자의 범위는 어디서부터 어디까지입니까?
> **정답:** 0부터 9까지

**문항 4.** 이번 실습은 인공지능 모델 구현뿐만 아니라 어떤 분야와의 연계를 목적으로 합니까?
> **정답:** 피지컬 컴퓨팅

---

## 3. 심화 학습을 위한 서술형 과제

**질문 1. CNN(합성곱 신경망)의 역할과 특징에 대해 서술하시오.**
> **모범 답안:** CNN은 이미지의 공간적 정보를 보존하면서 특징을 효과적으로 추출할 수 있는 신경망이다. 단순한 수치 데이터가 아닌 손글씨와 같은 이미지 데이터를 처리할 때 탁월한 성능을 발휘하며, MNIST 실습에서는 숫자의 형태적 특징을 학습하여 0~9 사이의 값을 분류하는 역할을 수행한다.

**질문 2. 손글씨 숫자 인식 모델을 피지컬 컴퓨팅 장치와 결합했을 때 기대할 수 있는 효과를 설명하시오.**
> **모범 답안:** 소프트웨어 내부에서만 작동하던 AI 모델을 피지컬 컴퓨팅 장치와 결합하면 현실 세계의 데이터를 직접 입력받아 처리할 수 있다. 예를 들어, 카메라로 찍은 실물 손글씨를 인식하여 하드웨어(로봇 팔, 디스플레이 등)를 제어하는 등 인공지능의 실용적인 활용 범위를 넓힐 수 있다.

## 4. 주요 용어 사전

| 용어 | 정의 및 설명 |
| :--- | :--- |
| **MNIST** | 인공지능 학습을 위해 널리 사용되는 손글씨 숫자 이미지 데이터셋. 0부터 9까지의 숫자로 이루어져 있다. |
| **CNN (Convolutional Neural Network)** | 이미지 처리에 특화된 신경망 구조로, 합성곱(Convolution) 연산을 통해 이미지의 특징을 찾아내는 인공지능 모델의 한 종류이다. |
| **합성곱 신경망** | CNN의 국문 명칭으로, 이미지의 픽셀 정보를 분석하여 패턴을 학습하는 인공 신경망이다. |
| **피지컬 컴퓨팅 (Physical Computing)** | 디지털 기기가 현실 세계의 아날로그 정보를 감지하고, 이를 처리하여 물리적인 결과를 출력하는 컴퓨팅 방식이다. |
| **인공지능 모델 (AI Model)** | 데이터를 학습하여 특정 목적(예: 숫자 인식)을 수행할 수 있도록 만들어진 소프트웨어 결과물이다. |
