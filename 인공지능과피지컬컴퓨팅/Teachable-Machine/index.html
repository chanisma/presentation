⚡ 2년 전 vs 지금

이제 우리는 CNN의 원리를 알고 있습니다

2년 전의 나

  • AI가 사진을 보고 "학습"한다
  • 데이터를 많이 주면 좋다
  • 어떻게 되는지는 모르지만 신기하다

지금의 나

  • CNN이 필터로 특징을 추출한다
  • 컨볼루션 → 풀링 → 분류 과정을 안다
  • Softmax가 확률을 계산해 분류한다

이제 Teachable Machine 내부에서 무슨 일이 벌어지는지 이해할 준비가 되었습니다.

🎨 비유로 이해하기

처음부터 학습 (CNN 직접 훈련)

그림을 전혀 모르는 사람에게 "선이 뭔지", "색이 뭔지", "형태가 뭔지"부터 가르친 후, "이건 고양이, 이건 강아지"를 구분하게 하는 것

◆ 수십만 장의 데이터 필요

◆ 몇 시간~며칠의 학습 시간

전이학습 (Teachable Machine)

이미 그림 실력이 뛰어난 화가에게 "이 꽃은 장미, 이 꽃은 튤립이야"라는 새로운 분류 기준만 알려주는 것

✦ 클래스당 수십 장이면 충분

✦ 몇 초~몇 분이면 학습 완료

적은 데이터로도 빠르게 학습이 가능한 이유 — MobileNet이 이미 "보는 법"을 알고 있기 때문입니다.

📸 Step 1 — 클래스 만들기 & 데이터 수집

클래스 이름 정하기

"Class 1", "Class 2" 이름을 의미 있게 바꾸세요

예: "주먹", "보", "가위"

"클래스 추가"로 새 클래스 생성

웹캠으로 데이터 수집

"웹캠" 버튼 → "길게 눌러 녹화" 클릭

클래스당 최소 50장 이상 촬영

다양한 각도와 거리에서 촬영하세요

데이터 수집 팁

  • 손을 살짝 움직이면서 촬영하면 다양한 데이터 확보
  • 배경이 다른 곳에서도 추가 촬영하면 정확도 향상
Teachable Machine

⚙ Step 2 — 학습시키기 & 테스트

  1. 01

    "모델 학습시키기" 클릭

    전이학습이 시작됩니다 — MobileNet의 특징 추출기 위에 우리 분류층이 학습됩니다

  2. 02

    학습 과정 관찰

    Epoch(반복 횟수)와 Loss(오차)가 줄어드는 것을 확인하세요

  3. 03

    실시간 테스트

    웹캠 앞에서 각 동작을 해보며 인식률을 확인합니다

학습이 몇 초 만에 끝나는 이유? MobileNet의 특징 추출 레이어는 이미 완성되어 있고, 우리는 마지막 분류층만 학습시키기 때문입니다.
