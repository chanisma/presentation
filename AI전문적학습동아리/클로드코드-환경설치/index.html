1단계 — Visual Studio Code 설치
🖥️

Visual Studio Code 설치

코드 편집기 + 한국어 팩 + Live Server 확장 설치
STEP 1

VSCode 다운로드 및 설치

인터넷에서 "visual studio code" 검색 후 공식 사이트에서 다운로드하여 설치합니다.

STEP 2

한국어 팩 설치

왼쪽 패널 확장(Extensions) 클릭 →
"Korean Language Pack" 검색 → 설치
오른쪽 아래 RESTART 버튼이 뜨면 클릭!

STEP 3

Live Server 확장 설치

확장 검색창에 "Live Server" 입력 후 설치합니다.
나중에 만든 HTML 파일을 바로 브라우저로 볼 때 사용합니다.

STEP 4

터미널(커맨드 창) 열기

VSCode 우측 상단 패널 설정/해제 버튼 클릭
→ 아래 검정 창이 생기면
→ 창 위 오른쪽 ▼ 화살표 클릭
→ "Command Prompt" 선택

💡 팁: 아래쪽에 생긴 검정 화면이 커맨드 창(터미널)입니다.
앞으로 여기에 명령어를 입력할 거예요.
🔍 확장 아이콘 위치: 왼쪽 세로 메뉴에서 네모가 4개인 아이콘을 찾으세요.
2단계 — Claude Code 설치
🤖

Claude Code 설치 및 실행

커맨드 창에 한 줄만 붙여넣으면 자동 설치됩니다
STEP 1 — 설치 명령어 실행

커맨드 창(터미널)에 아래 내용을 복사하여 붙여넣고 Enter

curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd

설치가 자동으로 진행됩니다. 완료까지 잠시 기다려 주세요.

⚠️
환경변수 추가 안내
설치 완료 후 "~~~ is not in your PATH"라는 문구가 보이면 환경변수를 추가해야 합니다!
환경변수 추가 방법 (필요한 경우만)

① 키보드 윈도우 버튼 누르고 path 검색

→ "시스템 환경 변수 편집" 클릭

② 환경 변수 → Path 더블클릭 → 새로 만들기

→ 위 메시지에서 ~~~에 표시된 경로를 그대로 붙여넣기 → 확인

STEP 2 — Claude Code 실행

커맨드 창에 claude 입력 후 Enter

처음 실행 시 설정 화면이 나옵니다.
모르는 항목은 기본값(Enter)으로 선택하면 됩니다.

💡 실행이 안 될 때: VSCode를 완전히 끄고 다시 켜면 해결되는 경우가 많습니다.
🔑 로그인: Claude 계정이 필요합니다. claude.ai에서 먼저 가입해 주세요.
3단계 — Git & GitHub Desktop 설치
🌿

Git & GitHub Desktop 설치

파일을 저장하고 관리하는 도구 — Claude Code가 작업 파일을 저장할 때 필요합니다

🌿 Git 설치

STEP 1 — Git 다운로드

아래 주소 접속 후 다운로드

https://git-scm.com/install/windows

"Click here to download" 클릭

STEP 2 — 설치 진행

설치 중 에디터 선택 화면에서

"Use Visual Studio Code as Git's default editor" 선택
나머지는 모두 Next(다음) 클릭

🐙 GitHub Desktop 설치

STEP 3 — GitHub Desktop 다운로드

아래 주소 접속 후 다운로드

https://desktop.github.com/download/
STEP 4 — 설치 및 로그인

GitHub 계정으로 로그인

GitHub 계정이 없으면 github.com에서 먼저 가입합니다.
가입 후 GitHub Desktop에서 로그인하면 됩니다.

💡 GitHub는 작업 파일을 인터넷에 저장하는 공간입니다.
구글 드라이브와 비슷한 개념이에요.
3단계 — GitHub 리파지토리(작업 공간) 설정
📁

GitHub 리파지토리(작업 공간) 설정

리파지토리 = 작업 파일들이 저장되는 폴더
처음 만드는 경우
새 리파지토리 생성
  • ①
    GitHub Desktop에서 Add 클릭 상단 메뉴에서 Add 버튼 클릭
  • ②
    Create new repository 선택
  • ③
    Repository name에 원하는 이름 입력 예: presentation (영어, 공백 없이)
  • ④
    Create repository 클릭
기존 것을 가져오는 경우
리파지토리 복제(Clone)
  • ①
    Add 클릭 → Clone repository 선택
  • ②
    목록에서 가져올 리파지토리 선택
  • ③
    Clone 클릭
🖥️ VSCode에서 바로 열기:
GitHub Desktop에서 현재 리파지토리 이름을 오른쪽 클릭
→ "Open in Visual Studio Code" 선택
4단계 — Presentation Builder 설치
📊

Presentation Builder 스킬 설치

파일 하나를 지정된 폴더에 복사하면 설치 완료!
STEP 1 — 스킬 파일 받기

presentation-builder.md 파일 다운로드

아래 버튼을 클릭하여 스킬 파일을 다운로드합니다.

⬇ presentation-builder.md 다운로드
STEP 2 — 폴더에 파일 복사

아래 위치에 파일을 복사합니다

C:\users\{사용자 이름}\.claude\commands

{사용자 이름}은 컴퓨터마다 다릅니다.
예: C:\users\teacher01\.claude\commands

⚠️ .claude 폴더가 안 보일 때: 파일 탐색기 상단 메뉴 → 보기 → "숨긴 항목" 체크 활성화
STEP 3 — Claude Code 재시작

VSCode를 완전히 끄고 다시 켜기

VSCode를 종료 후 다시 실행하고
커맨드 창에 claude 입력하여 다시 시작합니다.

STEP 4 — 설치 확인

Claude Code에서 아래 명령어 입력

/presentation-builder 안녕하세요

응답이 오면 설치 성공입니다! 🎉

5단계 — Presentation Builder 사용 방법
✨

발표 자료 만들기

명령어 한 줄로 아름다운 HTML 프레젠테이션 자동 생성!
기본 사용법
/presentation-builder 수업 관련 설명을 여기에 적어주세요

📝 활용 예시

예시 1 — 간단하게
/presentation-builder 수학 1학년 함수 개념 수업 자료
예시 2 — 자세하게
/presentation-builder 국어 3학년 시 해석 수업. 학습 목표: 은유법 이해하기 대상: 고등학교 3학년

🎨 디자인 추가하기

방법 1 — 새로운 디자인 요청

텍스트로 설명하기

presentation-builder에 새로운 디자인 추가: (디자인 설명)
방법 2 — 기존 자료 참고

PDF 파일 활용하기

기존 발표 자료를 PDF로 저장 후:

@C:\users\user\desktop\파일명.pdf 참고해서 디자인 추가
파일 확인 방법 — Live Server로 결과 보기
🌐

만든 파일 브라우저로 확인하기

Live Server를 이용하면 클릭 한 번으로 바로 확인!
파일 저장 위치
프레젠테이션은 이 경로에 저장됩니다
과목명 / 수업명 / index.html
📂 예시:
수학/함수개념/index.html
국어/시해석수업/index.html
영어/독해전략/index.html

🖱️ Live Server로 열기

  • ①
    VSCode 왼쪽 파일 탐색기에서 index.html 찾기 과목명 → 수업명 → index.html
  • ②
    index.html 파일에서 오른쪽 클릭
  • ③
    "Open with Live Server" 클릭 브라우저가 열리면서 프레젠테이션이 표시됩니다!
