Fundamentum

✦ AI에 수학이 필요한 이유

Mathematics as the Language of Intelligence

인공지능은 본질적으로 수학적 모델입니다. 데이터를 숫자로 표현하고, 패턴을 수식으로 찾고, 최적의 답을 계산으로 구합니다.

데이터 → 숫자

이미지는 픽셀 값의 행렬, 텍스트는 벡터로 변환됩니다. 모든 데이터는 수학적 형태로 바뀌어야 AI가 처리할 수 있습니다.

패턴 → 수식

AI는 데이터 속 규칙을 함수(f(x))로 표현합니다. "고양이 사진을 인식한다"는 것은 수학 함수를 찾는 과정입니다.

학습 → 최적화

AI의 학습은 손실함수를 최소화하는 미분 과정입니다. 경사하강법으로 최적의 가중치를 찾아갑니다.

판단 → 확률

"이 사진이 고양이일 확률 97%"처럼, AI의 모든 판단은 확률과 통계에 기반합니다.

Functio

✦ 함수 — AI의 기본 언어

Input → Process → Output

AI 모델은 거대한 함수입니다. 입력(x)을 받아 출력(y)을 내놓는 규칙, 즉 y = f(x)를 데이터에서 스스로 찾아냅니다.

선형 함수

y = wx + b

가장 기본적인 신경망 연산. 가중치(w)와 편향(b)으로 입력을 변환합니다.

시그모이드 함수

σ(x) = 1 / (1 + e⁻ˣ)

출력을 0~1 사이로 압축하여 "확률"로 해석합니다. 이진 분류에 핵심적입니다.

ReLU 함수

f(x) = max(0, x)

음수는 0, 양수는 그대로 통과. 딥러닝에서 가장 많이 쓰이는 활성화 함수입니다.

Matrix et Vector

✦ 행렬과 벡터 — 데이터의 표현

Linear Algebra for AI

벡터 (Vector)

  • 숫자들의 1차원 나열 — 방향과 크기를 가진 화살표
  • 단어를 벡터로: "왕" = [0.2, 0.8, 0.1, ...]
  • 유사한 의미 → 벡터가 가까이 위치
  • 벡터 연산: 왕 - 남자 + 여자 ≈ 여왕

행렬 (Matrix)

  • 숫자들의 2차원 배열 — 표(table) 형태
  • 28×28 손글씨 이미지 = 784개 숫자 행렬
  • 신경망의 가중치도 행렬로 저장
  • 행렬 곱셈 = 신경망의 핵심 연산

GPU가 AI에 강한 이유: 행렬 곱셈을 동시에 수천 개씩 병렬 처리할 수 있기 때문입니다.

Probabilitas

✦ 확률과 통계 — 불확실성 다루기

Probability & Statistics

조건부 확률

P(A|B) = P(A∩B) / P(B)

스팸 메일 필터링의 핵심. "이 단어가 있을 때 스팸일 확률"을 계산하는 베이즈 정리의 기초입니다.

정규분포

N(μ, σ²)

자연 현상의 대부분은 정규분포를 따릅니다. AI가 "정상"과 "이상"을 구분하는 기준이 됩니다.

Softmax 함수

P(i) = eˣⁱ / Σeˣʲ

신경망 출력을 확률 분포로 변환합니다. 10개 숫자 인식 시 각 숫자일 확률의 합은 항상 1입니다.

최대우도추정

θ* = argmax L(θ|data)

관측된 데이터를 가장 잘 설명하는 모델 파라미터를 찾는 통계적 방법입니다.

Calculus

✦ 미분 — AI가 학습하는 원리

Gradient Descent

미분은 "변화율"을 구하는 도구입니다. AI는 미분으로 "어떤 방향으로 가중치를 바꿔야 오차가 줄어드는지"를 알아냅니다.

  • ※

    기울기(Gradient)란?

    함수의 각 변수에 대한 편미분 값의 모음. 다차원 공간에서 "가장 가파르게 올라가는 방향"을 가리킵니다.

  • ※

    경사하강법의 비유

    안개 낀 산에서 가장 낮은 곳을 찾는 것과 같습니다. 발밑의 경사(기울기)를 느끼고, 가장 가파르게 내려가는 방향으로 한 걸음씩 이동합니다.

  • ※

    가중치 업데이트 공식

    w ← w − η · ∂L/∂w

    학습률(η) × 기울기만큼 가중치를 반대 방향으로 조정합니다. 이 과정을 수만 번 반복하면 AI가 "학습"됩니다.

Optimum

✦ 손실함수와 최적화

Loss Function & Optimization

MSE 평균제곱오차
CE 교차엔트로피
η 학습률

손실함수란?

AI의 예측값과 정답 사이의 오차를 측정하는 함수입니다. 회귀 문제에는 MSE, 분류 문제에는 교차엔트로피를 사용합니다.

학습률의 역할

한 걸음의 보폭을 결정합니다. 너무 크면 최적점을 지나치고, 너무 작으면 학습이 매우 느려집니다.

AI 학습의 목표: 손실함수의 값을 최소로 만드는 가중치 조합을 찾는 것. 이것이 "최적화"입니다.

Exempla

✦ 실생활 속 AI 수학 사례

Real-world Applications

얼굴 인식

얼굴의 특징점을 128차원 벡터로 변환하여 벡터 간 거리를 계산합니다. 거리가 가까우면 같은 사람으로 판단합니다.

음성 비서

음파를 주파수 행렬(스펙트로그램)로 변환한 뒤, 확률 모델로 가장 가능성 높은 단어 조합을 찾아냅니다.

자율주행

카메라 영상에 합성곱 연산을 적용해 차선·보행자를 인식하고, 확률적 예측으로 안전한 경로를 계산합니다.

우리가 매일 사용하는 검색엔진, SNS 피드, 번역기, 카메라 필터 모두 선형대수 + 확률 + 미분의 조합으로 작동합니다.
