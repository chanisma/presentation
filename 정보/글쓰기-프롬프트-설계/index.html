이 수업이 중요한 이유

AI에게 "잘 물어보는 능력"이 곧 AI 활용 능력. 같은 AI도 프롬프트에 따라 결과가 완전히 달라집니다.

프롬프트 개념

✦ 프롬프트란?

AI에게 보내는 지시문 — 질문의 질이 답변의 질을 결정합니다

AI 글쓰기 도구

ChatGPT, Claude, Gemini 등 대형언어모델(LLM) 기반 도구들

지난 수업에서 배운 LLM을 활용합니다

프롬프트 (Prompt)

AI에게 입력하는 텍스트 지시문. 무엇을, 어떻게 해달라는 요청 전체를 말합니다.

핵심 원리

좋은 프롬프트 → 좋은 결과

구체적·명확·구조화된 입력일수록 원하는 결과가 나옵니다

글쓰기에서 AI가 잘 하는 것

  • 공식 문체로 내용 정리·다듬기
  • 키워드를 문단으로 확장하기
  • 다양한 표현으로 바꿔 쓰기
  • 분량에 맞게 조절하기

AI가 못 하는 것 (내가 해야 할 것)

  • 내가 실제로 한 활동 내용 파악
  • 내가 느낀 점·배운 점 진술
  • 구체적인 수치·날짜·결과 제공
  • 교과별 특수 맥락 설명
프롬프트 비교

◆ 좋은 프롬프트 vs 나쁜 프롬프트

같은 AI에게 같은 주제를 물어도 결과가 완전히 다릅니다

나쁜 프롬프트
"자기평가문 써줘"
  • 어떤 활동을 했는지 설명 없음
  • 내가 배운 것·느낀 점 없음
  • 길이·문체·형식 지정 없음
  • 결과: 누구에게나 해당하는 공허한 문장
좋은 프롬프트
[역할] 나는 고2 학생이야.
[맥락] 정보 수업에서 AI 이미지 생성 프로젝트를 했어. 프롬프트를 3번 수정하며 원하는 이미지에 가까워지는 걸 배웠어.
[과제] 이 활동에 대한 자기평가문을 작성해줘.
[형식] 200자 이내, 1인칭, 활동 내용·배운 점·느낀 점 포함.

→ 나만의 경험이 담긴 자기평가문 초안 생성!

핵심 차이: AI는 내 경험을 모른다. 내가 무엇을 했는지 구체적으로 알려줄수록 결과가 좋아집니다.
프롬프트 4요소

✦ 프롬프트 설계 4요소

이 4가지를 채우면 좋은 프롬프트가 완성됩니다

① 역할 (Role)

내가 누구인지 AI에게 알려주기

예: "나는 고2 학생이야" "나는 ○○고 2학년이야"

② 맥락 (Context)

어떤 수업에서 무슨 활동을 했는지 설명

예: "1학기 수학 수업에서 통계 데이터 분석 프로젝트를 했어. 팀장으로 발표를 이끌었어."

③ 과제 (Task)

AI에게 원하는 것을 명확하게 지시

예: "이 활동에 대한 자기평가문을 작성해줘" "자기평가문 초안을 만들어줘"

④ 형식 (Format)

결과물의 길이·문체·스타일 지정

예: "200자 이내, 3인칭으로" "학교생활기록부 문체로, 띄어쓰기 포함 500자"

4요소를 모두 채울수록 더 좋은 결과!  |  처음에는 태그 [역할] [맥락] [과제] [형식]을 그대로 붙여 쓰고, 각 항목을 채워보세요.
예시 제시 기법

◆ 예시를 주면 AI가 따라합니다 — One-shot / Few-shot

원하는 결과물의 형식·문체를 보여주면 AI가 그 스타일로 작성합니다

Zero-shot

예시 없이 지시만

→ AI가 일반적인 형식으로 작성. 기대와 다를 수 있음

One-shot

원하는 결과 예시 1개 제공

→ AI가 예시의 길이·문체·구조를 파악하여 비슷하게 작성

Few-shot

좋은 예시 2~3개 제공

→ 패턴을 파악하여 더 일관된 결과 생성

One-shot 프롬프트 예시 ([형식]에 예시 추가)

[역할] 나는 고2 학생이야.
[맥락] 정보 수업에서 AI 이미지 생성 프로젝트를 했어. 프롬프트를 3번 수정하며 원하는 결과에 가까워지는 걸 배웠어.
[과제] 자기평가문을 작성해줘. 아래 예시와 같은 문체·길이로.
[예시] "수학 수업에서 통계 프로젝트를 수행하며 데이터 뒤에 숨겨진 의미를 읽어내는 능력이 중요함을 깨달았다. 단순히 수치를 계산하는 데서 그치지 않고…"
[형식] 1인칭, 200자 이내.
절차 설명 기법

✦ 생각하는 순서를 알려주세요 — 절차 설명 기법

AI에게 단계별 작성 순서를 지시하면 구조적인 결과를 얻습니다

절차 없이
[과제] 자기평가문을 작성해줘.

→ AI가 임의 순서로 작성. 배운 점이 빠지거나 활동 나열에 그칠 수 있음

절차 포함
[과제] 다음 순서로 자기평가문을 작성해줘:
① 내가 한 활동을 1~2문장으로 요약
② 그 과정에서 배운 점·역량을 서술
③ 느낀 점과 앞으로의 다짐으로 마무리

→ 구조가 잡힌 완성도 높은 자기평가문 생성

절차 설명이 특히 유용한 경우: 여러 항목을 모두 포함해야 할 때  |  특정 순서로 내용이 전개되어야 할 때  |  AI가 중요한 내용을 빠뜨릴 때
자기평가문이란?

◆ 자기평가문이란?

내 학습 과정과 성장을 내가 직접 기록하는 글

자기평가문 개념

수업 활동·학습 경험을 스스로 돌아보고, 배운 점·느낀 점·성장 과정을 1인칭으로 서술하는 글

자기평가문의 구성

  • 활동 내용 — 무엇을 했나
  • 배운 점 — 어떤 지식·역량을 얻었나
  • 느낀 점 — 과정에서 어떤 생각을 했나
  • 성장·다짐 — 앞으로 어떻게 할 것인가

언제 쓰나?

  • 수행평가 자기평가 항목
  • 포트폴리오 소개 글
  • 프로젝트 마무리 성찰
  • 활동 보고서 소감

AI가 도움이 되는 부분

내가 메모한 키워드와 경험을 입력하면 → 자연스러운 문장으로 정리·확장·다듬기

주의사항

AI 초안은 방향만 잡아주는 도구. 실제 느낀 점·구체적 사실은 반드시 내 언어로 수정·보완해야 합니다.

활동 → 프롬프트 구조화

✦ 실제 활동을 자기평가문 프롬프트로 만드는 4단계

내가 한 활동을 구조화하면 좋은 프롬프트가 됩니다

1

평가할 활동 선택

이번 학기 수업 중 기억에 남는 활동 1개 선택 → 수행평가 발표 / 실험·실습 / 독서 토론 / 프로젝트 등 자유롭게

2

활동 내용 메모 (4가지)

무엇을 했나 (활동명·소재)  |  어떻게 했나 (방법·나의 역할)  |  배운 점 (지식·역량)  |  느낀 점 (생각·깨달음·아쉬운 점)

3

4요소 프롬프트 작성

[역할] 나는 고2 학생  |  [맥락] 2단계에서 메모한 내용  |  [과제] 자기평가문 작성  |  [형식] 200자 이내, 1인칭, 활동 내용·배운 점·느낀 점 포함

4

AI 초안 검토 & 내 언어로 수정

AI 초안 확인 → 사실과 다른 부분·과장된 표현 수정 → 내가 실제로 느낀 점을 구체적으로 보완 → 필요 시 재요청

자기평가문 프롬프트 예시

◆ 완성된 자기평가문 프롬프트 예시

아래 예시를 참고하여 내 활동에 맞게 바꿔보세요

국어 수업 예시
[역할] 나는 고2 학생이야.
[맥락] 국어 수업에서 소설 '파친코'를 읽고 팀 토론을 했어. 조장 역할을 맡아 토론 방향을 이끌었고, 처음에는 의견 조율이 어려웠지만 적극적으로 발언을 정리하면서 팀이 결론을 낼 수 있었어.
[과제] 이 활동에 대한 자기평가문을 작성해줘.
[형식] 200자 이내, 1인칭, 활동 내용·배운 점·느낀 점 포함.
정보 수업 예시
[역할] 나는 고2 학생이야.
[맥락] 정보 수업에서 AI 이미지 생성 프로젝트를 했어. 프롬프트를 3번 수정하며 원하는 이미지에 가까워지는 걸 경험했어. 처음엔 막막했는데 구체적으로 쓸수록 좋은 결과가 나온다는 걸 깨달았어.
[과제] 이 활동에 대한 자기평가문을 작성해줘.
[형식] 200자 이내, 1인칭, 활동·배운 점·느낀 점 포함.
맥락이 핵심! 활동명만 쓰지 말고 내 역할·어려웠던 점·깨달은 것을 구체적으로 넣을수록 진짜 나다운 자기평가문이 나옵니다.
초안 다듬기

◆ AI 초안, 왜 잘 다듬어야 할까?

AI 출력물은 출발점 — 진짜 글은 다듬는 과정에서 완성됩니다

AI 글은 '평균'이다

AI는 수백만 개 글의 평균값. 내 실제 경험의 감동과 구체적인 맥락은 내가 직접 채워야 합니다.

진정성이 평가의 핵심

평가자는 학생의 진짜 성장이 담긴 글을 원합니다. 내 목소리가 없는 글은 설득력을 잃습니다.

사실 오류 점검 필수

AI는 내가 알려준 맥락만 압니다. 날짜·수치·활동 내용을 잘못 표현하거나 과장할 수 있습니다.

글쓰기 역량은 내 것

프롬프트 작성 → 초안 생성 → 내 언어로 수정. 이 전 과정이 나의 AI 글쓰기 역량이 됩니다.

AI는 초안 생성 도구, 완성은 내 몫. "내 언어로 수정"이 가장 중요한 단계입니다.
실습

✦ 실습: 내 자기평가문 프롬프트 직접 만들기

교과·활동은 자유롭게 선택 — 기본 4요소 + 고급 기법까지 적용해보세요

1

평가할 활동 1개 선택

이번 학기에 기억에 남는 수업 활동 1개 선택 — 국어·수학·영어·사회·과학·정보·예체능 자유롭게

2

활동 내용 메모 (4가지)

무엇을 했나  |  내 역할·방법은  |  무엇을 배웠나  |  어떤 점을 느꼈나 — 짧은 키워드라도 구체적으로

3

기본 4요소 프롬프트 작성

[역할] [맥락: 2번 내용] [과제: 자기평가문 작성해줘] [형식: 1인칭, 200자, 활동·배운 점·느낀 점 포함]

4

고급 기법 1개 선택하여 추가

예시 제시: [과제]에 "아래 예시 문체로 작성해줘" + 예시 문장 1개 추가  |  절차 설명: [과제]에 "① 활동 요약 → ② 배운 점 → ③ 느낀 점·다짐 순서로 작성해줘" 추가

5

AI 입력 → 초안 검토 & 내 언어로 수정

AI 초안 확인 → 과장·오류 수정 → 실제 느낀 점을 내 말로 보완 → 결과가 마음에 안 들면 맥락 추가해서 재요청

오늘의 핵심 정리

  • ✅ 프롬프트 = AI에게 보내는 지시문. 구체적일수록 좋은 결과
  • ✅ 설계 4요소: [역할] 내가 누구  |  [맥락] 무슨 활동  |  [과제] 자기평가문 작성  |  [형식] 1인칭·200자
  • ✅ 자기평가문: AI는 초안만, 실제 느낀 점·구체적 사실은 반드시 내 언어로 수정
  • ✅ 맥락이 핵심: 활동명만 쓰지 말고 내 역할·어려웠던 점·깨달은 것까지 담기
  • ✅ 마음에 안 들면 더 구체적인 정보를 추가해서 다시 요청하기

프롬프트 설계 능력은 AI 시대의 핵심 역량입니다.
