Google Fonts

✦ Google Fonts로 폰트 바꾸기

무료 웹 폰트를 3단계로 적용하세요

1

fonts.google.com 접속

원하는 폰트를 검색하고 "Get embed code" 클릭

2

<head> 안, <style> 위에 link 태그 붙여넣기

<head> <link href="https://fonts...?family=Jua"> <!-- 여기! --> <style> ... </style> </head>
3

CSS에서 font-family 적용

body { font-family: 'Jua', sans-serif; }
추천 한글 폰트: Noto Sans KR (깔끔) · Jua (귀여움) · Black Han Sans (임팩트)
폰트 속성 — 코드를 수정하면 오른쪽이 바뀝니다

✶ 글자를 다루는 CSS 속성들

/* 글자 크기 (px, rem) */ .font-demo { font-size: 24px; }
/* 글자 굵기 (100~900, bold) */ .font-demo { font-weight: 700; }
/* 줄 간격 (1.5~1.8 추천) */ .font-demo { line-height: 1.7; }
/* 자간 — 글자 사이 간격 */ .font-demo { letter-spacing: 0.05em; }

색상 — 코드를 수정하면 오른쪽이 바뀝니다

♦ 색상 표현 3가지

/* 1. 이름으로 지정 */ .color-demo h2 { color: tomato; }
/* 2. HEX 코드 (#RRGGBB) */ .color-demo p { color: #3498db; }
/* 3. rgb(빨강, 초록, 파랑) */ .color-demo span { color: rgb(142, 68, 173); }

▲ 색상값을 바꿔보세요!

background — 코드를 수정하면 오른쪽이 바뀝니다

❀ background 활용하기

/* 단색 배경 */ .bg-demo-1 { background-color: #4ECDC4; }
/* 그라데이션 (방향, 색1, 색2) */ .bg-demo-2 { background: linear-gradient(135deg, #667eea, #764ba2); }
/* 배경 이미지 + 꽉 채우기 */ .bg-demo-3 { background-image: url('https://upload.wikimedia.org/wikipedia/commons/thumb/b/b6/Image_created_with_a_mobile_phone.png/640px-Image_created_with_a_mobile_phone.png'); background-size: cover; }

hover 효과 — 코드를 수정하고 오른쪽 카드에 마우스를 올려보세요

♢ hover 효과로 인터랙션 추가

/* 부드러운 전환 (필수!) */ .hover-card { transition: all 0.3s ease; } /* 마우스 올리면 변화 */ .hover-card:hover { transform: scale(1.05); box-shadow: 0 8px 24px rgba(0,0,0,0.12); border-color: #4ECDC4; }
transition 없이 :hover만 쓰면 뚝딱 바뀜!
