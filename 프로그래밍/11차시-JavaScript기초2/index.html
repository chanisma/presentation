반복문 — for문 구조
STEP 1 · 반복문

◆ for문 기본 구조

같은 코드를 원하는 횟수만큼 자동으로 반복합니다

for (let i = 0; i < 5; i++) { console.log(i); // 0, 1, 2, 3, 4 순서로 출력 }

초기값

let i = 0

반복 시작 전 딱 한 번만 실행
보통 변수 i를 0으로 초기화

조건식

i < 5

true이면 계속 반복
false가 되면 반복 종료

증감식

i++

한 번 반복할 때마다 실행
i++는 i = i + 1과 동일

STEP 1 · 반복문

✦ for문 실습 예제

3 × 1 = 3 3 × 2 = 6 3 × 3 = 9 ...
반복문 — while문
STEP 1 · 반복문

◆ while문 — for문과 비교

조건이 true인 동안 반복 — 횟수가 정해지지 않을 때 유용합니다

for문
// 반복 횟수가 정해진 경우 for (let i = 1; i <= 5; i++) { console.log(i); } // → 1, 2, 3, 4, 5
while문
// 같은 결과를 while로 let i = 1; while (i <= 5) { console.log(i); i++; } // → 1, 2, 3, 4, 5
구분for문while문
언제?반복 횟수가 정해진 경우조건에 따라 유동적인 경우
초기값/증감괄호 안에 한 줄로 작성따로 직접 작성 필요
주의사항거의 없음무한루프 주의!
결론: 반복 횟수를 미리 알면 for, 조건이 변수에 따라 달라지면 while — 대부분의 경우 for가 더 안전합니다

함수 (Function)
STEP 2 · 함수

✦ 함수란?

재사용 가능한 코드를 하나의 묶음으로 만드는 것

자판기 비유

버튼을 누르면 → 음료가 나온다

함수 이름을 호출하면 → 코드가 실행된다

레시피 비유

재료(매개변수)를 넣으면 → 요리(결과값)가 나온다

같은 레시피로 여러 번 요리 가능!

함수가 없을 때 — 코드 중복

// 같은 코드가 3번 반복! console.log("안녕 김철수"); console.log("안녕 이영희"); console.log("안녕 박민준");

↓ 함수로 만들면

function greet(name) { console.log("안녕 " + name); } greet("김철수"); greet("이영희"); greet("박민준");
함수 — 선언과 호출
STEP 2 · 함수

◆ 함수 선언과 호출

function 키워드로 함수를 만들고, 이름을 불러 실행합니다

function add(a, b) { return a + b; }

함수 이름

add

호출할 때 사용하는 이름
동사로 짓는 것이 관례

매개변수

a, b

함수에 전달하는 입력값
여러 개 가능, 없어도 됨

반환값 (return)

a + b

결과를 돌려줌
없으면 undefined 반환

// 함수 호출 — 숫자를 전달하면 합계가 돌아옴 let result1 = add(3, 5); // result1 = 8 let result2 = add(10, 20); // result2 = 30 console.log(result1, result2); // 8 30
함수 — 실습 예제
STEP 2 · 함수

✦ 함수 실습 예제

두 가지 함수를 직접 작성하고 콘솔에서 실행해보세요

예제 1 두 수의 합 반환
function add(a, b) { return a + b; } let x = add(3, 7); let y = add(10, 25); console.log(x); // 10 console.log(y); // 35
▶ 콘솔 출력
10
35
예제 2 인사말 출력 함수

2학년 김지은님, 안녕!
함수는 한 번 만들면 몇 번이든 재사용 가능! 같은 코드를 여러 번 쓰지 않아도 됩니다
함수 — 필요성
STEP 2 · 함수

◉ 왜 함수를 써야 할까?

코드 반복 vs 함수 사용 — 무엇이 달라지는지 직접 비교합니다

✗ 함수 없이 (BAD)
// 세금 계산 — 3번 중복! let p1 = 10000 * 1.1; console.log("상품1:", p1); let p2 = 25000 * 1.1; console.log("상품2:", p2); let p3 = 8500 * 1.1; console.log("상품3:", p3); // 세율 변경? → 3곳 모두 수정!
✓ 함수 사용 (GOOD)
// 세금 계산 함수 1개만! function withTax(price) { return price * 1.1; } console.log("상품1:", withTax(10000)); console.log("상품2:", withTax(25000)); console.log("상품3:", withTax(8500)); // 세율 변경? → 함수 1곳만!
함수를 쓰면 — 수정이 쉬워지고, 오류가 줄어들고, 코드를 이해하기 쉬워집니다

JS로 할 수 있는 것

  • 색상, 크기 등 스타일 변경
  • 새 요소 추가 / 삭제

✦ 요소 선택하기

HTML 요소를 먼저 선택(가져와야) 내용을 바꿀 수 있습니다

document.getElementById("id명")
id 속성으로 요소 1개를 정확하게 가져옵니다 — 가장 빠르고 정확
document.querySelector("CSS선택자")
CSS 선택자 문법으로 요소를 가져옵니다 — 첫 번째 일치하는 요소만 반환
<!-- HTML --> <h1 id="title">안녕하세요</h1> <p class="desc">설명 텍스트</p> <button id="btn">클릭</button>
// JavaScript — 요소 선택 let title = document.getElementById("title"); let desc = document.querySelector(".desc"); // 클래스 let btn = document.querySelector("#btn"); // id console.log(title); // <h1 id="title">...</h1>

◆ 요소 내용과 스타일 바꾸기

선택한 요소의 텍스트, 스타일, HTML을 자바스크립트로 변경합니다

.textContent

가장 안전한 방법

.style.속성

backgroundColor, fontSize

let el = document.getElementById("title"); // 텍스트 변경 el.textContent = "반갑습니다!"; // 스타일 변경 (CSS: color → JS: color, background-color → backgroundColor) el.style.color = "#FF6B6B"; el.style.fontSize = "2rem"; el.style.backgroundColor = "#1A1A2E"; // HTML 변경 (강조 태그 포함) el.innerHTML = "<strong>강조된</strong> 텍스트";

이벤트 (Event)
STEP 4 · 이벤트

◆ 이벤트란?

사용자의 행동(클릭, 입력, 스크롤 등)을 감지하여 반응하는 것

마우스 이벤트

  • click — 클릭
  • dblclick — 더블클릭
  • mouseover — 마우스 올림

키보드 이벤트

  • keydown — 키 누름
  • keyup — 키 뗌
  • input — 값 변경 시

폼 이벤트

  • submit — 폼 제출
  • change — 선택 변경
  • focus — 포커스

이벤트의 흐름

사용자 행동 → 브라우저가 감지 → 등록된 함수(핸들러) 실행

이 수업에서 배울 것

addEventListener로 이벤트를 감지하고 원하는 동작을 연결합니다

이벤트 — addEventListener
STEP 4 · 이벤트

✦ addEventListener 사용법

요소에 이벤트를 등록하는 가장 현대적인 방법

요소.addEventListener("이벤트 종류", 실행할 함수);

요소 (Element)

btn, el, ...

getElementById나 querySelector로 선택한 요소

이벤트 종류

"click"

문자열로 전달
"click", "input", "keydown" 등

실행할 함수

function() {}

이벤트 발생 시 실행
익명 함수로 바로 작성

let btn = document.getElementById("myBtn"); btn.addEventListener("click", function() { console.log("버튼이 클릭되었습니다!"); });
