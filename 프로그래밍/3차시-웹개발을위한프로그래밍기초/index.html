태그(Tag)란?

📦

상자 형태의 구조

웹 페이지의 각 요소를 상자처럼 직관적으로 표현할 수 있도록 만든 표현 방식

🔧

각자의 역할

각각의 태그는 고유한 역할 혹은 기능을 지니고 있음

태그 = 웹 페이지를 구성하는 상자
각 상자마다 이름과 역할이 정해져 있습니다
#HTML #태그 #기초
HTML태그기초

태그의 기본 구조

태그는 시작 태그와 종료 태그로 구성되어 있으며, 상자의 테두리처럼 동작하여 안에 내용을 보관할 수 있습니다.

<태그명>
→
내용
←
</태그명>
⬆ 시작 태그 종료 태그 ⬆
코드
<p>This is a test document.</p>
결과

시작 태그 <p>로 열고, 종료 태그 </p>로 닫습니다. 그 사이에 내용이 들어갑니다.
#HTML #태그 #기초
HTML태그기초

태그의 중첩 (Nesting)

태그의 내용에 또 다른 태그가 들어갈 수 있으며, 이를 통해 더 세밀하게 웹 사이트를 구조화 할 수 있습니다.

<div> 바깥 상자
<p> 안쪽 상자
내용이 여기에!
<div> <p> 내용이 여기에! </p> </div>
상자 안의 상자! 큰 상자 안에 작은 상자를 넣을 수 있는 것처럼, 태그 안에 태그를 넣어 복잡한 구조를 만들 수 있습니다.
#HTML 문서의 기본적인 구조
HTML문서 구조

HTML 문서의 기본 구조

<html> <head> ... 세부정보 ... </head> <body> ... 실제 콘텐츠 ... </body> </html>
🏠 <html> = 집 전체
🏷️ <head> = 지붕 (보이지 않는 정보)

페이지 타이틀, CSS, JS 연결 등

👁️ <body> = 거실 (보이는 내용)

화면에 띄울 글, 이미지 등

🏠 집 비유: html은 집 전체, head는 지붕(보이지 않는 정보), body는 거실(눈에 보이는 내용)입니다.

태그의 종류를 학습할 때 기억할 것

태그의 종류는 너무나 다양하기 때문에, 모든 태그의 기능과 역할을 익히는데 초점을 맞추기 보다는 태그가 뼈대의 역할을 한다는 것과 작성 방법에 익숙해지는 것에 초점을 맞춰 연습하는게 좋습니다.

실습: 개인 프로필 페이지 만들기

나만의 프로필 페이지를 HTML로 직접 만들어 봅시다

주제: 나, 또는 내가 좋아하는 것 (게임, 아이돌, 음식 등 자유)

1
기본 구조 입력
html, head, body 뼈대를 작성하세요
<html>
  <head></head>
  <body></body>
</html>
2
<title>로 페이지 제목 넣기
<title>홍길동의 프로필</title>
3
<h1>, <h2>로 제목과 소제목 쓰기
큰 제목 1개 + 소제목 2~3개를 만들어 보세요
4
<p>로 자기소개 또는 설명 글 쓰기
각 소제목 아래에 설명 문단을 추가하세요
5
<ul>, <li>로 목록 만들기
좋아하는 것, 취미 등을 리스트로 나열해 보세요

🏆 도전 과제

<ol>을 사용해서 순위가 있는 목록(예: 좋아하는 음식 TOP 3)도 만들어 보세요!
