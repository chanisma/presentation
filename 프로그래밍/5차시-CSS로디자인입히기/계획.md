## 4차시 10~13페이지 연결 계획

### 페이지 10: 블록 레벨 vs 인라인 레벨
- **4차시 내용**: 블록(div, p, h1, ul)은 새 줄에서 시작하며 가로 전체 차지, 인라인(a, img, strong, span)은 줄바꿈 없이 나란히 배치
- **5차시 활용**:
  - CSS `display` 속성으로 블록/인라인 전환하는 실습
  - `width`, `height`, `margin`, `padding`이 블록/인라인에서 다르게 동작함을 시연
  - 4차시의 벽돌(블록) vs 글자(인라인) 비유를 이어받아 CSS 박스모델로 확장

### 페이지 11: class와 id 속성
- **4차시 내용**: id는 유일한 이름(페이지에 하나), class는 그룹 이름(여러 요소에 동일 적용 가능). CSS 맛보기로 `.btn` 스타일 적용 전/후 비교
- **5차시 활용**:
  - CSS 선택자 핵심 개념으로 직접 연결 (`.class`, `#id` 선택자)
  - 4차시 맛보기에서 보여준 버튼 스타일을 직접 작성하는 실습
  - 하나의 class로 여러 요소에 동시 스타일 적용하는 효율성 체험

### 페이지 12: 이 외의 태그들
- **4차시 내용**: button, input 등 추가 태그 존재 안내, W3Schools 레퍼런스 소개
- **5차시 활용**:
  - CSS로 button, input 등의 기본 스타일을 커스터마이징하는 예시 (선택적)
  - "태그는 구조, CSS는 디자인"이라는 역할 분리 강조
  - 필요하면 W3Schools CSS 레퍼런스도 함께 안내

### 페이지 13: 프로젝트 진행 (HTML 뼈대 세우기)
- **4차시 내용**: ① 뼈대 세우기 ② 콘텐츠 넣기 ③ div로 구분하기 / "배치가 안 되는 게 정상"이라는 안내
- **5차시 활용**:
  - 학생들이 만든 HTML 뼈대에 CSS를 입히는 것이 이번 차시의 핵심 실습
  - "4차시에서 배치가 안 됐던 이유 = CSS가 없었기 때문" → 동기부여
  - div로 구분해둔 영역에 배경색, 여백, 정렬 등을 적용하며 레이아웃 완성
