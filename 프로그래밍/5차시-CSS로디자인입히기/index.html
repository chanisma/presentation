#HTML #class #id
HTMLclass, id 속성

식별자 속성: class와 id

특정 요소를 구분하여 활용하기 위해 이름을 지정하는 속성

id — 유일한 이름

특정 요소를 정확하게 구분하기 위한 식별자

<div id="header">...</div>

→ 한 페이지에 딱 한 번만 사용 가능

class — 그룹 이름

같은 스타일이나 기능을 모아서 적용하기 위한 식별자

<button class="btn-primary">...</button>

→ 여러 요소에 중복해서 같은 이름 사용 가능

✨ 이 이름표(class, id)들이 바로 오늘 CSS 디자인을 입힐 때 대상을 지목하는 핵심 역할(선택자)을 하게 됩니다!
#CSS #개념

CSS 란?

Cascading Style Sheets

HTML이 웹 문서의 뼈대와 내용을 만든다면, CSS는 그 뼈대에 옷을 입히고 예쁘게 화장하는 디자인 역할을 합니다.

  • 웹 문서의 폰트 크기, 글자 색상, 배경색 등을 지정
  • 요소들의 화면 배치(Layout)와 간격을 좌우함
  • 화면 크기에 따라 모양이 변하는 반응형 웹을 구현

#CSS #기초문법
CSS문법과 구조

CSS 코드는 어떻게 생겼을까?

h1 {
color: blue;
font-size: 24px;
}
선택자
(Selector)

누구를 꾸밀 것인가?

위 예시에서는 문서의 모든 <h1> 태그를 지목했습니다.

속성
(Property)

어떤 속성을 바꿀 것인가?

예시는 글자 색(color)과 크기(font-size)를 변경합니다.

값
(Value)

어떻게 변경할 것인가?

파란색(blue)으로, 24픽셀 크기로 꾸밉니다.
중요: 한 구문이 끝나면 반드시 세미콜론(;)을 붙입니다!

CSS를 적용하는 3가지 방법

HTML 문서에 스타일을 입히는 방식은 크게 3가지입니다.
📝

1. 인라인 스타일

HTML 태그 안에 직접 style 속성 작성

가장 간단하지만 유지보수 힘듦
📄

2. 내부 스타일시트

HTML 문서의 <head> 안 <style> 태그에 작성

하나의 파일 안에서 해결
📁

3. 외부 스타일시트

별도의 .css 파일을 만들고 HTML 문서에 연결

현업 필수! 깔끔한 유지보수

👉 이제부터 1번부터 차근차근 직접 실습해 봅시다!

#CSS #1번째방법
방법 1

인라인 스타일 (Inline Style)

HTML 태그 내부에 style 속성을 사용하여 디자인을 직접 입력하는 방법

🧑‍💻 코드 실습 (수정 가능) 📋 복사하기
<h1 style="color: red; background-color: yellow;">
  인라인 스타일입니다!
</h1>
특징: 직관적이고 빠르지만, 꾸며줄 태그가 많아질수록 코드 관리가 감당할 수 없을 만큼 지저분해집니다.
👁️ 눈에 보이는 결과

인라인 스타일입니다!

#CSS #2번째방법
방법 2

내부 스타일 시트 (Internal Style Sheet)

HTML의 <head> 부분에 <style> 태그를 열어 한 번에 작성하는 방법

<head> <style>
h1 { color: blue; }

p { font-size: 20px; }
</style> </head>
<body>
<h1>파란색 제목</h1>
<p>조금 커진 문장입니다.</p>
</body>

👁️ 화면 결과 (실시간 렌더링)

파란색 제목

조금 커진 문장입니다.

장점: HTML 태그 자체가 깔끔해지고, 선택자를 통해 한 번에 여러 태그의 디자인 변경이 가능합니다.
#CSS #3번째방법
방법 3

외부 스타일 시트 (External Style Sheet) - ★실무 권장

디자인 코드만 담긴 .css 파일을 따로 만들고, HTML에 연결(<link>)하는 방법

📄 index.html

문서 <head>에 링크 추가하기

<head>
  <link rel="stylesheet" href="style.css">
</head>

🎨 style.css (새 파일)

CSS 코드는 이곳에 분리 작성

body {
  background-color: #f0f0f0;
}

h1 {
  color: green;
}

가장 완벽한 분업! 프로젝트의 모든 페이지에서 같은 style.css를 링크하면,
수백 개의 페이지 디자인을 CSS 파일 단 하나 수정만으로 변경할 수 있습니다. 🪄
