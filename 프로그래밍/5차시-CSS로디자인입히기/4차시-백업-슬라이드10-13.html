블록 레벨 vs 인라인 레벨

📦 블록 레벨 (Block)

  • 항상 새로운 줄에서 시작
  • 가로 너비 전체를 차지
  • 수직으로 쌓임, 크기 조정 가능
<div> <p> <h1> <ul>

📝 인라인 레벨 (Inline)

  • 텍스트 흐름 중간에 삽입
  • 줄 바꿈 없음, 나란히 배치
  • 내용 크기에 맞게 고정됨
<a> <img> <strong> <span>

실제 렌더링 비교

블록 요소: 위아래로 쌓임 (벽돌)

<div> 첫 번째
<p> 두 번째
<h2> 세 번째

인라인 요소: 옆으로 나란히 (글자)

텍스트 <a> 사이에 <strong> 들어가고 <span> 줄바꿈 없이 나란히
#HTML #class, id 속성
HTMLclass, id

식별자 속성: class와 id

특정 요소를 구분하여 활용하기 위해 이름을 지정하는 속성

id — 유일한 이름

특정 요소를 정확하게 구분하기 위한 식별자

<div id="header">...</div>

→ 페이지에 하나만 사용

class — 그룹 이름

같은 속성을 적용하기 위한 식별자

<button class="btn">...</button>

→ 여러 요소에 같은 이름 가능
