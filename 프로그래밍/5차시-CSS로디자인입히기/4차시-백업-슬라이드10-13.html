#HTML #class, id 속성
HTMLclass, id

식별자 속성: class와 id

특정 요소를 구분하여 활용하기 위해 이름을 지정하는 속성

id — 유일한 이름

특정 요소를 정확하게 구분하기 위한 식별자

<div id="header">...</div>

→ 페이지에 하나만 사용

class — 그룹 이름

같은 속성을 적용하기 위한 식별자

<button class="btn">...</button>

→ 여러 요소에 같은 이름 가능
