transition 심화

✦ transition 속성 파헤치기

어떤 속성을, 얼마나 걸려서, 어떤 느낌으로 바꿀지 설정합니다

/* transition: 속성 시간 효과; */ .card { transition: all 0.3s ease; }
/* 속성별로 따로 지정 */ .card { transition: background 0.3s ease, transform 0.5s ease-out; }
/* timing 함수 종류 */ ease /* 기본, 자연스러운 감속 */ linear /* 일정한 속도 */ ease-in /* 천천히 시작 */ ease-out /* 천천히 끝남 */

transform 변환

✶ transform으로 요소를 변환하기

이동, 크기 변경, 회전을 CSS만으로 구현합니다

/* 이동 — translate(x, y) */ .move { transform: translateX(50px); }
/* 크기 — scale(배율) */ .grow { transform: scale(1.2); }
/* 회전 — rotate(각도) */ .spin { transform: rotate(45deg); }

⬡ transform + hover 실전 조합

여러 transform을 한 번에 쓸 수 있습니다

.profile-card { transition: all 0.3s ease; } .profile-card:hover { transform: translateY(-8px) scale(1.02); box-shadow: 0 12px 28px rgba(0,0,0,0.15); }
transform을 여러 개 쓸 때는 한 줄에 공백으로 구분합니다

❖ @keyframes — CSS 애니메이션 만들기

hover 없이도 자동으로 움직이는 애니메이션

/* 1. 애니메이션 정의 */ @keyframes bounce { 0% { transform: translateY(0); } 50% { transform: translateY(-20px); } 100% { transform: translateY(0); } } /* 2. 요소에 적용 */ .bouncing { animation: bounce 1s ease infinite; }
bounce

↑ 자동으로 통통 튀고 있습니다

속성설명예시 값
animation-name이름bounce
animation-duration시간1s
animation-timing-function효과ease
animation-iteration-count반복infinite

✦ 실전 애니메이션 3가지

깜빡이기 (blink)

blink
@keyframes blink { 0% { opacity: 1; } 50% { opacity: 0.2; } 100% { opacity: 1; } }

흔들기 (shake)

shake
@keyframes shake { 0%,100% { translateX(0); } 25% { translateX(-8px); } 75% { translateX(8px); } }

펄스 (pulse)

pulse
@keyframes pulse { 0%,100% { scale(1); } 50% { scale(1.2); } }
