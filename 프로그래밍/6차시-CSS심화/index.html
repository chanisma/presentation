1. class와 id란?
개념 설명

class와 id란?

특정 요소만 골라서 CSS를 적용하고 싶을 때 사용합니다.

.class (클래스)

여러 요소에 같은 이름을 붙일 수 있어요.

비유: "학년" - 1학년이 여러 명일 수 있듯이!

.highlight { color: red; }

#id (아이디)

하나의 요소에만 붙이는 고유한 이름이에요.

비유: "출석번호" - 딱 한 명만 가지고 있어요!

#special { background: gold; }
구분classid
CSS 선택자.이름 (점)#이름 (샵)
사용 가능 개수여러 요소에 중복 가능한 페이지에 하나만
비유"학년" (여러 명)"출석번호" (한 명)

2. 자손 선택자와 자식 선택자
개념 설명

자손 선택자와 자식 선택자

자손 선택자 A B

A 안에 있는 모든 B를 선택 (손자, 증손자 포함)

비유: "우리 가족 중 모든 아이들"

자식 선택자 A > B

A 바로 아래에 있는 B만 선택 (직계 자식만)

비유: "내 자녀만"

HTML 구조 트리:

<ul class="menu"> <li>메뉴 1</li> ← 자식 <li>메뉴 2 ← 자식 <ul> <li>서브메뉴 A</li> ← 자손 <li>서브메뉴 B</li> ← 자손 </ul> </li> <li>메뉴 3</li> ← 자식 </ul>

빨간색 = .menu > li (자식만)
파란색 = .menu li (모든 자손)

3. 박스 모델의 핵심
개념 설명

박스 모델의 핵심: Margin & Padding

margin (바깥 여백)
border (테두리)
padding (안쪽 여백)
content
(내용)

padding = 액자 안 여백 (그림과 틀 사이)

margin = 액자 사이 간격 (액자와 액자 사이)

방향 지정하기

margin-top : 위쪽
margin-right : 오른쪽
margin-bottom : 아래쪽
margin-left : 왼쪽

단축 표기법

margin: 10px; 상하좌우 모두 10px
margin: 10px 20px; 상하 10px, 좌우 20px
margin: 10px auto; 상하 10px, 좌우 자동 (가운데 정렬)
