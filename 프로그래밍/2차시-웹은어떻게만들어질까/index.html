인터넷 vs 웹
🌍

인터넷과 웹, 같은 걸까?

많이 헷갈리지만 사실은 다릅니다!
🛣️
인터넷 (Internet)
전 세계 컴퓨터를 연결하는
거대한 네트워크(도로)

1969년에 시작된
컴퓨터 간 통신 기술
≠
🌐
웹 (WWW)
인터넷 위에서 동작하는
서비스 중 하나(버스)

1991년에 만들어진
문서를 공유하는 시스템
💡 쉽게 이해하기
인터넷은 전국의 고속도로와 같아요. 도로 자체는 여러 교통수단이 다닐 수 있죠.
웹은 그 도로 위를 달리는 버스 같은 거예요. 도로 위에는 버스(웹) 말고도 택시(이메일), 오토바이(메신저), 트럭(파일 전송) 등 다양한 서비스가 있어요!
📌 핵심 정리
인터넷 = 네트워크(기반 시설) / 웹 = 인터넷 위의 서비스 중 하나. 우리가 브라우저로 보는 것이 바로 웹!
웹사이트가 보이는 과정 ①
☕

웹사이트는 어떻게 보일까?

카페에서 커피 주문하는 것과 같아요!

☕ 카페 주문

🙋‍♀️ 나 (손님)

"아이스 아메리카노 주세요!" → 주문

⬇️
👨‍🍳 바리스타

커피를 만들어서 전달 → 응답

🌐 웹사이트 접속

💻 클라이언트 (내 브라우저)

"naver.com 보여줘!" → 요청(Request)

⬇️
🖥️ 서버 (네이버 컴퓨터)

HTML/CSS/JS 파일 전달 → 응답(Response)

🔑 핵심! 웹사이트는 어딘가에 있는 서버 컴퓨터에 저장된 파일을 내 브라우저가 받아서 보여주는 것이에요!

단계별로 살펴보기

URL을 입력하면 무슨 일이 일어날까?

💡 이것도 알아두세요!
URL = 웹사이트의 주소 (예: https://www.naver.com)
DNS = 인터넷의 전화번호부. "naver.com"이라는 이름을 서버의 실제 주소(숫자)로 바꿔주는 역할
브라우저 = 크롬, 사파리, 엣지 같은 앱. 서버에서 받은 파일을 예쁘게 화면에 그려주는 프로그램
⚡ 놀라운 사실
이 모든 과정이 0.5초 이내에 일어나요! 우리가 주소를 입력하고 엔터를 누르면, 순식간에 전 세계 어딘가의 서버에서 파일을 가져옵니다.
웹페이지의 3가지 언어 ①

집을 짓는 것처럼!

웹페이지를 만드는 3가지 언어를 집 짓기에 비유해 봐요
🏗️
HTML
설계도 & 골조
방, 벽, 문, 창문의 위치를 정하는 것처럼
웹페이지의 구조와 내용을 담당

"여기에 제목, 여기에 사진,
여기에 버튼을 놓겠다!"
🎨
CSS
인테리어 디자인
벽지 색, 가구 배치, 조명을 꾸미는 것처럼
웹페이지의 디자인과 레이아웃을 담당

"배경은 파란색, 글자는 크게,
버튼은 둥글게!"
⚡
JavaScript
전자기기 & 자동화
스위치를 누르면 불이 켜지는 것처럼
웹페이지의 동작과 상호작용을 담당

"버튼을 누르면 메뉴가 열리고,
스크롤하면 이미지가 바뀌고!"
🎓 우리 수업에서는?
HTML부터 시작해서 → CSS로 꾸미고 → JavaScript로 움직이게 만드는 순서로 배웁니다!
한 번에 다 배우는 게 아니라, 차근차근 하나씩 배워요.
웹페이지의 3가지 언어 ②
👀

실제 코드는 이렇게 생겼어요

아직 외울 필요 없어요! 눈으로만 구경해 봅시다

🏗️ HTML (구조)

<h1>안녕하세요!</h1>
<p>반갑습니다</p>
<button>클릭</button>

"무엇을" 보여줄지 결정

🎨 CSS (디자인)

color: blue;
font-size: 24px;
background: pink;

"어떻게" 보여줄지 결정

⚡ JavaScript (동작)

if (버튼 클릭) {
  alert('안녕!');
}

"어떻게 반응"할지 결정

🏗️ 뼈대(HTML) + 🎨 옷(CSS) + ⚡ 두뇌(JS) = 🌐 웹사이트!
세 가지가 합쳐져서 우리가 보는 웹페이지가 완성됩니다
💬 걱정 마세요!
지금은 코드가 낯설어 보이지만, 다음 시간부터 하나하나 배우면 금방 익숙해져요. 이번 학기가 끝나면 여러분도 이런 코드를 직접 작성하게 됩니다!

📋 활동 순서
  • 1
    좋아하는 웹사이트 1개를 선택하세요
    멜론, 올리브영, 무신사, 쿠팡 등 무엇이든!
  • 2
    눈으로 먼저 헤더 / 내비 / 메인 / 푸터 영역을 찾아보세요
  • 3
    F12를 눌러 개발자 도구를 열고, 각 영역 위에 마우스를 올려 코드에서 어떤 태그인지 확인하세요
  • 4
    활동지에 각 영역의 내용과 태그 이름을 적어보세요
  • 5
    모둠원과 함께 분석 결과를 비교해 봅시다
