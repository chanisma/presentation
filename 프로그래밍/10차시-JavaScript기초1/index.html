STEP 2 · script 태그

✦ HTML에 JavaScript 넣는 방법

<script> 태그를 사용해서 HTML 파일 안에 JavaScript 코드를 작성합니다

방법 1 HTML 파일 안에 직접 작성
<body> <h1>내 페이지</h1> <p>안녕하세요!</p> <!-- ✓ </body> 바로 위에 작성 --> <script> let name = "철수"; console.log(name); </script> </body>
방법 2 별도 .js 파일로 연결
<!-- HTML 파일 --> <body> <h1>내 페이지</h1> <script src="app.js"></script> </body> // app.js 파일 (별도 파일) let name = "철수"; console.log(name);
왜 </body> 바로 앞에 넣나요? — HTML이 먼저 모두 화면에 그려진 뒤 JS가 실행되어야 오류가 줄어듭니다. 이 수업에서는 방법 1(직접 작성)을 사용합니다.

변수 — 자료형(Data Type)
STEP 3 · 변수

◆ 변수에 담을 수 있는 3가지 기본 타입

상자 안에 어떤 종류의 값을 넣느냐에 따라 자료형이 결정됩니다

문자열 (String)

큰따옴표 또는 작은따옴표로 감싼 텍스트

let name = "김철수"; let city = "전주"; let msg = '안녕!';

숫자 (Number)

따옴표 없이 그냥 쓴 숫자 (정수·소수 모두)

let age = 17; let score = 95.5; let year = 2026;

불리언 (Boolean)

참(true) 또는 거짓(false) 딱 두 가지

let isStudent = true; let isPassed = false; let isAdult = true;
숫자 17과 문자열 "17"은 다릅니다! 따옴표가 있으면 문자, 없으면 숫자 — 헷갈리지 마세요

◆ 산술 연산자 — 계산하기

수학 계산을 JavaScript로 표현합니다 — 결과를 console.log로 바로 확인하세요

기본 연산 + − * / % **
console.log(10 + 3); // 13 console.log(10 - 3); // 7 console.log(10 * 3); // 30 console.log(10 / 3); // 3.333... console.log(10 % 3); // 1 (나머지) console.log(2 ** 10); // 1024 (2의 10제곱)

let name = "철수"; let age = 17; console.log("이름: " + name); // 이름: 철수 console.log("나이: " + age + "세"); // 나이: 17세
▶ 주의!
"5" + 3 = "53" — 문자+숫자는 결합!

**

+

STEP 4 · 연산자

✦ 비교 연산자 — 결과는 항상 true/false

두 값을 비교해서 참(true) 또는 거짓(false)을 반환합니다

연산자의미예시결과
==같다(느슨)5 == "5"true
===같다(엄격)5 === "5"false
!=다르다(느슨)5 != 3true
!==다르다(엄격)5 !== "5"true
>크다10 > 5true
<작다3 < 1false
>=크거나 같다5 >= 5true
<=작거나 같다4 <= 3false

== vs === 핵심 차이

5 == "5" // true // 값만 비교 (타입 무시) 5 === "5" // false // 값 + 타입 모두 비교
실제 코드에서는 === 사용 권장! ==은 예상치 못한 결과를 낼 수 있어요
연산자 — 논리 연산자
STEP 4 · 연산자

◆ 논리 연산자 — 조건을 조합하기

여러 조건을 연결하거나 반전할 때 사용합니다 — 일상 언어로 생각하면 쉽습니다!

&& (AND)

둘 다 true일 때만 true
"그리고"

true && true // true true && false // false

|| (OR)

하나라도 true면 true
"또는"

true || false // true false || false // false

! (NOT)

true ↔ false 반전
"아닌"

!true // false !false // true

일상 비유로 이해하기

  • 비가 오고(&&) 우산이 없으면 → 우산 빌리기
  • 토요일이거나(||) 일요일이면 → 주말이다
  • 배가 고프지 않다(!isHungry) → 밥 안 먹기
let age = 17; let hasId = true; console.log(age >= 18 && hasId); // false (나이 미달) console.log(age >= 14 || hasId); // true (둘 중 하나 참)

STEP 5 · 조건문

✦ if문 — 만약 ~라면 ~해라

조건이 참(true)일 때만 중괄호 안의 코드가 실행됩니다

▶ if문 구조 분석

실행 부분

중괄호 { } 안의 코드만 실행됩니다

let score = 85; if (score >= 60) { console.log("합격!"); }

조건문 — if-else (두 갈래 분기)
STEP 5 · 조건문

◆ if-else — 그렇지 않으면

조건이 참이면 if 블록, 거짓이면 else 블록이 실행됩니다 — 반드시 둘 중 하나가 실행!

if-else 구조
let score = 85; if (score >= 60) { // 조건이 true일 때 console.log("합격"); } else { // 조건이 false일 때 console.log("불합격"); }
▶ score = 85일 때
합격

실습: 점수 → 합격/불합격

score 변수 값을 직접 바꿔가며 실행해 보세요
40이면? → 불합격 / 70이면? → 합격

핵심 정리

  • if와 else 중 반드시 하나만 실행
  • else는 단독 사용 불가, if 뒤에만 가능
  • 조건은 ( ) 안에, 실행 코드는 { } 안에
조건문 — else if (여러 갈래 분기)
STEP 5 · 조건문

✦ else if — 세 갈래 이상 분기

조건이 여러 개일 때 else if를 사용합니다 — 위에서부터 순서대로 확인합니다

학점 판별
let score = 78; if (score >= 90) { console.log("A"); // 90 이상 } else if (score >= 80) { console.log("B"); // 80~89 } else if (score >= 70) { console.log("C"); // 70~79 } else if (score >= 60) { console.log("D"); // 60~69 } else { console.log("F"); // 60 미만 }
▶ score = 78일 때
C
1

위에서부터 검사

score >= 90? NO → score >= 80? NO → score >= 70? YES! → "C" 출력

2

하나만 실행

조건이 참인 첫 번째 블록만 실행 후 나머지는 건너뜁니다

3

else는 마지막에

모든 조건이 false일 때 실행되는 기본값 역할

STEP 5 · 종합 실습

◆ 나이를 입력받아 분류하기

사용자에게 나이를 직접 입력받아 "어린이 / 청소년 / 성인"으로 판별합니다

전체 코드
// prompt()로 사용자 입력 받기 let input = prompt("나이를 입력하세요:"); let age = Number(input); // 문자→숫자 변환 if (age < 13) { console.log("어린이입니다."); } else if (age < 19) { console.log("청소년입니다."); } else { console.log("성인입니다."); }
▶ 나이 = 17 입력 시
청소년입니다.

새로 배운 것: prompt()

사용자가 텍스트를 입력할 수 있는 팝업창을 띄웁니다. 입력값은 항상 문자열!

Number() 변환

prompt는 문자열을 반환하므로 Number()로 숫자로 변환해야 크기 비교 가능!
