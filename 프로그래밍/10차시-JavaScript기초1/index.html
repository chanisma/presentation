<script> 태그

◆ HTML에서 JavaScript 작성하는 위치

<script> 태그 안에 JavaScript 코드를 작성합니다

방법 1 — 내부 방식 (이 수업에서 사용)

<body> <h1>내 페이지</h1> <p>내용...</p> <!-- body 맨 끝에 작성 ✓ --> <script> let name = "김철수"; console.log(name); </script> </body>

방법 2 — 외부 파일 연결

<!-- HTML 파일 --> <body> <h1>내 페이지</h1> <script src="app.js"></script> </body> // app.js 파일 (별도 파일) let name = "김철수"; console.log(name);

권장 위치: </body> 바로 앞

HTML이 먼저 모두 로딩된 뒤 JS가 실행 → 오류가 줄어듭니다. console.log()는 개발자 도구(F12) 콘솔 탭에서 확인합니다.

변수 (Variable)

✦ JavaScript 변수

데이터를 저장하는 그릇 — Python과 비교하며 배웁니다

Python 변수 선언
# 변수 선언 — 키워드 없이 대입 name = "김철수" age = 17 pi = 3.14 is_student = True # 값 변경 — 항상 가능 age = 18
JavaScript 변수 선언
// let — 나중에 값 변경 가능 let name = "김철수"; let age = 17; let isStudent = true; // const — 상수, 변경 불가 const PI = 3.14; // PI = 3; ← 오류 발생!
let 재할당 가능  |  const 한 번 정하면 변경 불가  |  문장 끝에 ; 세미콜론  |  불리언: Python True/False → JS true/false
연산자 (Operator)

◆ JavaScript 연산자

Python과 거의 동일합니다 — 기호 문법이 같아요!

산술 연산자

let a = 10, b = 3; a + b // 13 a - b // 7 a * b // 30 a / b // 3.33... a % b // 1 (나머지) a ** b // 1000 (거듭제곱)

비교 연산자

// 결과: true / false a > b // true a < b // false a >= 10 // true a == 10 // 값만 비교 a === 10 // 값+타입 비교 ✓ a !== b // true

논리 연산자

// Python → JavaScript // and → && // or → || // not → ! a>5 && b<5 // 둘 다 참 a>5 || b>9 // 하나만 참 !(a === b) // 반대

Python과 차이: 산술·비교 연산자는 동일! 논리 연산자만 달라요 → and→&&, or→||, not→!  |  === 권장 (값과 자료형을 함께 비교)

조건문 (Conditional)

✦ JavaScript 조건문

Python의 if / elif / else ↔ JavaScript의 if / else if / else

Python
score = 85 if score >= 90: grade = "A" elif score >= 80: grade = "B" elif score >= 70: grade = "C" else: grade = "F" print(grade) # B
JavaScript
let score = 85; let grade; if (score >= 90) { grade = "A"; } else if (score >= 80) { grade = "B"; } else if (score >= 70) { grade = "C"; } else { grade = "F"; } console.log(grade); // B
조건을 ( )로 감싸고  |  실행 코드를 { }로 묶기  |  elif → else if (두 단어)  |  들여쓰기 대신 중괄호가 블록을 구분합니다
종합 예제

◆ 종합 예제: 나이 계산기

변수 · 연산자 · 조건문을 모두 활용 — 내 개인 페이지에 바로 추가 가능!

JavaScript </body> 바로 앞에 작성
<script> // ① 변수 선언 let birthYear = 2008; let currentYear = 2026; // ② 연산자로 나이 계산 let age = currentYear - birthYear; // ③ 조건문으로 메시지 결정 let msg; if (age >= 19) { msg = "성인입니다"; } else if (age >= 16) { msg = "고등학생입니다"; } else { msg = "중학생 이하입니다"; } // ④ 화면에 출력 document.getElementById("result") .textContent = age + "세 · " + msg; </script>

HTML 구조

<!-- 결과를 표시할 요소 --> <p id="result">로딩 중...</p>

실행 결과

> result.textContent
18세 · 고등학생입니다

응용 아이디어

birthYear를 바꿔보세요!
2007 → 19세 · 성인입니다
2010 → 16세 · 고등학생입니다

오늘의 핵심 정리

  • ✅ JavaScript = HTML(뼈대) + CSS(디자인) + JS(동작) — 브라우저에서 바로 실행
  • ✅ <script> 태그를 </body> 바로 앞에 작성
  • ✅ let(변경 가능) / const(상수) 로 변수 선언, 문장 끝 ;
  • ✅ 논리 연산자: and → &&, or → ||, not → !
  • ✅ 조건문: if (조건) { } else if (조건) { } else { } — elif 대신 else if
