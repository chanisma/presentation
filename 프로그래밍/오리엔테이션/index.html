트랙 1 — 데이터 연구 프로그래밍

데이터 연구 프로그래밍

데이터로 세상을 읽고, 연구로 증명하기
"궁금한 것을 데이터로 답하는 연구자"
관심 주제에 대한 데이터를 직접 수집·분석하여 연구 보고서를 완성합니다

🔄 연구 프로세스

1

주제 선정

관심 분야에서 연구 질문 만들기

2

데이터 수집

공공데이터·설문·크롤링

3

분석 & 시각화

Python으로 그래프·통계 분석

📚 배울 도구
Python · pandas · matplotlib · Google Colab
💡 활용 예시
"우리 학교 학생의 수면 시간과 성적의 관계는?" · "전주시 미세먼지 추세 분석"

웹 언어

🏗️

HTML

CSS

웹 페이지의 디자인

<p style="color: blue">나의 첫 웹 페이지</p>
<button onclick="alert('클릭!')">눌러보세요</button>
🎓 최종 목표
자기소개 페이지 → 관심 주제 웹사이트 → 포트폴리오 사이트까지, 단계별로 완성합니다. 실제 인터넷에 배포하여 누구나 볼 수 있게 만듭니다!
