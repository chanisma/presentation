@media 쿼리

✦ @media 쿼리 기초

"화면이 이 크기 이하이면, 이 스타일을 적용해줘"

기본 문법

@media (max-width: 768px) { /* 768px 이하일 때 적용 */ .card-grid { grid-template-columns: 1fr; } }

자주 쓰는 브레이크포인트

/* 태블릿 */ @media (max-width: 768px) { } /* 모바일 */ @media (max-width: 480px) { }

핵심 포인트

@media는 CSS 안에서 "조건문" 역할을 합니다. 화면 너비에 따라 다른 스타일을 적용할 수 있어요.

미디어 쿼리 실전

◆ 3단계 브레이크포인트 전략

데스크톱 → 태블릿 → 모바일 순서로 스타일을 재정의합니다

데스크톱 (기본)

.cards { display: grid; grid-template-columns: repeat(3, 1fr); }

3열 그리드 배치

태블릿 (768px)

@media (max-width: 768px) { .cards { grid-template-columns: repeat(2, 1fr); } }

2열로 축소

모바일 (480px)

@media (max-width: 480px) { .cards { grid-template-columns: 1fr; } }

1열로 세로 배치

기본 스타일을 먼저 작성하고, @media로 작은 화면에서 바꿀 부분만 덮어씁니다.
유연한 레이아웃

✦ 유연한 레이아웃

고정 px 대신 상대 단위와 유연한 속성을 사용합니다

고정 vs 유연한 단위

/* ✗ 고정 — 화면을 넘침 */ .container { width: 960px; } /* ✓ 유연 — 화면에 맞춤 */ .container { width: 90%; max-width: 1100px; }

auto-fit으로 자동 배치

/* 카드가 화면에 맞게 자동 정렬 */ .card-grid { display: grid; grid-template-columns: repeat(auto-fit, minmax(260px, 1fr)); }

clamp() — 반응형 폰트의 비결

font-size: clamp(14px, 2vw, 22px) → 최소 14px, 최대 22px, 화면 비례 자동 조절

유연한 이미지

◆ 유연한 이미지

이미지가 화면 밖으로 넘치지 않고, 비율을 유지하며 줄어드는 방법

핵심 한 줄

/* 이미지가 부모보다 커지지 않음 */ img { max-width: 100%; height: auto; }

비율 고정 트릭

/* 16:9 비율 유지 컨테이너 */ .img-wrap { width: 100%; aspect-ratio: 16 / 9; overflow: hidden; } .img-wrap img { width: 100%; height: 100%; object-fit: cover; }

max-width: 100%

부모 너비를 넘지 않게 자동 축소

aspect-ratio

너비가 변해도 비율을 유지

object-fit: cover

잘리더라도 영역을 꽉 채움

실습

✦ 실습: 비반응형 → 반응형 변환

비반응형 데모 페이지의 코드를 반응형으로 바꿔봅시다

1

viewport 메타 태그 추가

<meta name="viewport" content="width=device-width, initial-scale=1.0">

2

고정 너비를 유연한 단위로 변경

width: 960px → width: 90%; max-width: 1100px

3

미디어 쿼리 추가

@media (max-width: 768px)에서 그리드를 1열로 변경

4

이미지를 유연하게 변경

고정 이미지 크기 → width: 100%; aspect-ratio: 16/9

오늘의 핵심 정리

  • ✅ 반응형 디자인 = 미디어 쿼리 + 유연한 레이아웃 + 유연한 이미지
  • ✅ @media (max-width: 768px)로 태블릿/모바일 스타일 분기
  • ✅ 고정 px 대신 %, vw, auto-fit, clamp() 사용
  • ✅ 이미지에 max-width: 100%와 aspect-ratio 적용
