HTML태그의 속성

태그의 속성 (Attribute)

태그에 따라, 부가적인 정보 혹은 설정이 필요한 경우 태그를 작성할 때 속성을 추가하여 사용할 수 있습니다.

<태그명 속성명="값" 속성명="값">
내용
</태그명>
코드
<img src="test.jpg" alt="테스트">
의미

src → 이미지 파일 경로
alt → 대체 텍스트

이미지 태그의 경우, 이미지 파일의 경로(src), 대체 텍스트(alt), 크기(width) 등의 설정을 할 수 있습니다.

HTML태그의 종류이미지

<img> 이미지 태그

이미지 삽입을 위한 태그 — 종료 태그가 없음!

속성이름설명
srcSource이미지 파일 경로 또는 URL
altAlternative이미지 표시가 안될 때 대체할 텍스트
width가로 길이px 단위 혹은 % 지정
height세로 길이px 단위 혹은 % 지정
코드
<img src="https://picsum.photos/200/150" alt="랜덤 이미지" width="200">

<a> 링크 태그

다른 페이지나 웹사이트로 이동하는 링크를 생성합니다.

이미지에 링크를 걸어보세요! <a> 안에 <img>를 넣으면 이미지를 클릭하면 다른 사이트로 이동합니다.

블록 레벨 vs 인라인 레벨

📦 블록 레벨 (Block)

  • 항상 새로운 줄에서 시작
  • 가로 너비 전체를 차지
  • 수직으로 쌓임, 크기 조정 가능
<div> <p> <h1> <ul>

📝 인라인 레벨 (Inline)

  • 텍스트 흐름 중간에 삽입
  • 줄 바꿈 없음, 나란히 배치
  • 내용 크기에 맞게 고정됨
<a> <img> <strong> <span>

실제 렌더링 비교

블록 요소: 위아래로 쌓임 (벽돌)

<div> 첫 번째
<p> 두 번째
<h2> 세 번째

인라인 요소: 옆으로 나란히 (글자)

텍스트 <a> 사이에 <strong> 들어가고 <span> 줄바꿈 없이 나란히
#HTML #class, id 속성
HTMLclass, id

식별자 속성: class와 id

특정 요소를 구분하여 활용하기 위해 이름을 지정하는 속성

id — 유일한 이름

특정 요소를 정확하게 구분하기 위한 식별자

<div id="header">...</div>

→ 페이지에 하나만 사용

class — 그룹 이름

같은 속성을 적용하기 위한 식별자

<button class="btn">...</button>

→ 여러 요소에 같은 이름 가능

CSS 적용 후

더 많은 태그를 알아보고 싶다면?

수업 시간에 다룬 태그 이외에도 수없이 많은 태그들이 존재합니다.
<button>, <input> 등 유용한 태그들이 많으나, 당장 사용되지 않을 내용이거나 CSS나 JS에서 처리할 수 있기 때문에 다루지 않았습니다.
