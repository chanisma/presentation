#HTML #태그의 속성
HTML태그의 속성

태그의 속성 (Attribute)

태그에 따라, 부가적인 정보 혹은 설정이 필요한 경우 태그를 작성할 때 속성을 추가하여 사용할 수 있습니다.

<태그명 속성명="값" 속성명="값">
내용
</태그명>
코드
<img src="test.jpg" alt="테스트">
의미

src → 이미지 파일 경로
alt → 대체 텍스트

이미지 태그의 경우, 이미지 파일의 경로(src), 대체 텍스트(alt), 크기(width) 등의 설정을 할 수 있습니다.
#HTML #태그의 종류
HTML태그의 종류이미지

<img> 이미지 태그

이미지 삽입을 위한 태그 — 종료 태그가 없음!

속성이름설명
srcSource이미지 파일 경로 또는 URL
altAlternative이미지 표시가 안될 때 대체할 텍스트
width가로 길이px 단위 혹은 % 지정
height세로 길이px 단위 혹은 % 지정
코드
<img src="https://picsum.photos/200/150" alt="랜덤 이미지" width="200">
결과
#HTML #태그의 종류
HTML태그의 종류링크

<a> 링크 태그

다른 페이지나 웹사이트로 이동하는 링크를 생성합니다.

속성이름설명
hrefHypertext Reference이동할 페이지 주소 (URL)
targetTarget (대상)"_blank"로 넣으면 새 탭에서 열림
코드
<a href="https://google.com" target="_blank"> 구글로 이동 </a>
결과 (클릭해 보세요!)
구글로 이동 ↗
💡 target="_blank"를 넣으면 새 탭에서 열리고, 빼면 같은 탭에서 이동합니다.
#HTML #태그의 종류
HTML태그의 종류구조

<div> 구조 구분 태그

Division — HTML의 구획을 구분하는 태그

📦

구획 나누기

특별한 스타일은 없고, 순수하게 구획을 나누기 위해 사용

🔗

요소 묶기

큰 틀에서 구획을 나누거나, 여러 요소를 묶을 때 사용

<div> <h1>프로필</h1> <p>안녕하세요!</p> </div> <div> <h2>취미</h2> <ul> <li>독서</li> <li>게임</li> </ul> </div>

실습: 프로필 페이지 업그레이드!

지난 시간에 만든 페이지에 이미지, 링크, 구조를 추가합시다
1
<div>로 구획 나누기
소개, 특징, 링크 등 섹션별로 <div>로 감싸서 구분하기
2
<img>로 이미지 넣기
<img src="사진.jpg" alt="설명" width="200">
3
<a>로 링크 연결하기
<a href="주소" target="_blank">이름</a>

🏆 도전 과제

<a> 안에 <img>를 넣어 이미지를 클릭하면 다른 사이트로 이동하도록 만들어보세요!
