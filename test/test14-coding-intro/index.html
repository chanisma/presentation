코딩이란?

코딩 = 명령어를 작성하는 것

코딩(Coding)은 컴퓨터가 이해할 수 있는 언어로 지시사항을 적는 행위입니다. 사람이 한국어로 말하듯, 컴퓨터는 프로그래밍 언어로 소통합니다.
  • 우리 주변의 앱, 게임, 웹사이트 모두 코딩으로 만들어짐
  • 문제를 논리적으로 생각하고 해결하는 능력을 기름
  • 누구나 배울 수 있는 현대의 필수 기술

알고리즘
ALGORITHM

알고리즘 = 문제를 푸는 순서

알고리즘은 어떤 문제를 해결하기 위한 단계별 절차입니다. 라면 끓이는 순서처럼, 컴퓨터에게 할 일을 순서대로 알려주는 것입니다.

일상 속 알고리즘

  • 라면 끓이기: 물 끓이기 → 면 넣기 → 수프 넣기 → 완성
  • 길 찾기: 출발 → 신호등 확인 → 방향 결정 → 도착
  • 검색하기: 키워드 입력 → 결과 확인 → 선택

알고리즘의 3요소

  • 순차 — 위에서 아래로 순서대로 실행
  • 조건 — 상황에 따라 다른 행동 (if/else)
  • 반복 — 같은 작업을 여러 번 (for/while)
