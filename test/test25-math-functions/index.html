01 — 함수의 정의
개념 도입

함수란 무엇인가

집합 X의 각 원소에 집합 Y의 원소가 하나씩만 대응할 때, 이 대응을 X에서 Y로의 함수라 한다.
f : X → Y   또는   y = f(x)
핵심 조건 X의 모든 원소에 대응이 존재해야 한다 — 대응이 없거나 2개 이상이면 함수가 아니다.
예 (함수 O) f(x) = 2x + 1  →  x마다 유일한 y값 존재
예 (함수 X) x² + y² = 1  →  하나의 x에 두 y값 대응
02 — 핵심 용어

정의역 · 공역 · 치역

Domain

정의역

함수의 입력값 전체 집합 X
모든 x가 대응 대상

Codomain

공역

출력값이 속할 수 있는 전체 집합 Y
치역을 포함하는 범위

Range

치역

실제로 대응된 y값들의 집합
공역의 부분집합: f(X) ⊆ Y

치역 ≠ 공역임에 주의하라.  f(x) = x²에서 공역이 R이라면 치역은 [0, ∞)이다.
치역 = { f(x) | x ∈ X }
03 — 함수의 종류

함수의 종류

  • 1:1

    일대일 함수 (단사함수)

    서로 다른 x에 서로 다른 f(x)가 대응. x₁ ≠ x₂ ⇒ f(x₁) ≠ f(x₂)

  • ON

    위로의 함수 (전사함수)

    치역 = 공역. Y의 모든 원소에 대응하는 X의 원소가 존재한다.

  • 1-1

    일대일 대응 (전단사함수)

    단사 + 전사를 동시에 만족. 역함수가 존재하는 조건이다.

  • ID

    항등함수 / 상수함수

    항등함수: f(x) = x  —  상수함수: f(x) = c (c는 상수, 치역 = {c})

04 — 그래프

함수의 그래프

수직선 판별법

  • 임의의 수직선이 그래프와 1번만 만나면 함수의 그래프
  • 2번 이상 만나는 지점이 있으면 함수가 아니다
  • 원 x²+y²=r²은 수직선과 2교점 → 함수 아님

그래프 해석

  • x 절편: f(x)=0이 되는 x값
  • y 절편: f(0) 값
  • 증가/감소 구간: 기울기의 부호로 판단
  • 대칭축: 우함수(y축), 기함수(원점)
05 — 합성 & 역함수

합성함수와 역함수

Composition

합성함수

두 함수를 순서대로 적용한 새로운 함수

(g ∘ f)(x) = g(f(x))

주의: g∘f ≠ f∘g (교환법칙 불성립)

Inverse

역함수

일대일 대응인 함수에만 존재

f ∘ f-1 = I   (I: 항등함수)

그래프는 y = x에 대해 대칭

역함수 구하는 절차: (1) y = f(x) 로 놓기 → (2) x에 대해 풀기 → (3) x와 y 교환 → (4) f⁻¹(x) 표기
