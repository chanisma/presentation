01 · 데이터 과학이란?

What is Data Science?

데이터 과학이란?

데이터 과학은 통계학, 컴퓨터 공학, 도메인 지식을 결합해 데이터에서 가치 있는 인사이트를 발견하는 학문입니다.
📊

통계학

데이터 패턴을 수학적으로 분석하고 의미 있는 결론을 도출합니다.

💻

컴퓨터 공학

대용량 데이터를 처리하고 알고리즘을 구현하는 기술적 토대입니다.

🔬

도메인 지식

분야별 전문 지식으로 데이터가 실제로 무엇을 의미하는지 파악합니다.

🤖

머신러닝

컴퓨터가 데이터로부터 스스로 규칙을 학습하는 핵심 기술입니다.

02 · 데이터의 종류

Types of Data

데이터의 종류

🔢

수치형 데이터

키, 몸무게, 온도 등 숫자로 표현되는 연속적 값

🏷️

범주형 데이터

성별, 혈액형처럼 이름으로 구분되는 그룹 데이터

📝

텍스트 데이터

SNS 게시글, 리뷰, 뉴스 기사 등 언어 기반 정보

🖼️

이미지 데이터

사진, 영상 등 픽셀로 이루어진 시각적 정보

📅

시계열 데이터

주가, 기온 변화처럼 시간 순서에 따른 데이터

🗺️

공간 데이터

GPS 좌표, 지도처럼 위치 정보를 담은 데이터

03 · 데이터 수집과 전처리

Data Collection & Preprocessing

데이터 수집과 전처리

  • 🌐

    데이터 수집

    공공 데이터 포털, API, 웹 스크래핑 등 다양한 방법으로 원시 데이터를 모읍니다.

  • 🧹

    데이터 정제

    결측값, 이상치, 중복 데이터를 찾아 제거하거나 보완하는 과정입니다.

  • ⚙️

    데이터 변환

    분석에 적합한 형태로 정규화, 인코딩, 스케일링 등을 수행합니다.

  • ✅

    데이터 검증

    전처리된 데이터가 분석 목적에 맞는 품질을 갖추었는지 확인합니다.

04 · 데이터 시각화

Data Visualization

데이터 시각화

"숫자보다 그래프가 훨씬 빨리 이해된다." — 복잡한 데이터를 한눈에 파악하게 해주는 시각화 기법들
📈

선 그래프

시간에 따른 변화 추이를 파악할 때 사용합니다.

📊

막대 그래프

항목별 수량을 비교하고 차이를 시각적으로 나타냅니다.

🔵

산점도

두 변수 간의 상관관계와 분포 패턴을 확인합니다.

🌡️

히트맵

색상의 농도로 데이터의 강도와 밀집도를 표현합니다.

05 · 머신러닝 기초

Machine Learning Basics

머신러닝 기초

  • 🎓

    지도학습 (Supervised Learning)

    정답이 있는 데이터로 학습. 분류(스팸 필터)와 회귀(가격 예측)가 대표적입니다.

  • 🔍

    비지도학습 (Unsupervised Learning)

    정답 없이 패턴을 스스로 발견. 군집화로 비슷한 데이터를 묶어냅니다.

  • 🎮

    강화학습 (Reinforcement Learning)

    보상과 벌칙으로 최적의 행동을 학습. 게임 AI와 자율주행에 활용됩니다.

  • 🧠

    딥러닝 (Deep Learning)

    뇌 신경망을 모방한 다층 구조로 이미지 인식, 언어 번역에서 탁월한 성능.
