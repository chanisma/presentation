# SKILL.md 개선 사항 요약

## 개선 항목 목록

### 1. ⚠️ section-icon 누락 방지 강화 (P1)
- **문제**: 4/5 프레젠테이션에서 section-icon 누락 (가장 빈번한 위반)
- **원인**: "모든 콘텐츠 슬라이드" 범위가 모호하여 TOC, 참고자료 등에서 누락
- **개선**:
  - 규칙에 ⚠️ 경고 마커 추가, "가장 빈번한 위반 항목" 표기
  - "section-icon이 없는 h2는 절대 사용하지 않는다" 명시
  - 예외 슬라이드 명시적 열거: cover, end만 예외 (TOC, 참고자료는 예외 아님)
  - CSS 값 허용 범위 명시: font-size 1.1~1.3em, margin-right 0.3~0.5em

### 2. 이모지 예산 규칙 명확화 (P1)
- **문제**: 리스트 내 이모지, cover/end 이모지 포함 여부 불명확
- **개선**:
  - 이모지 정의 추가: Unicode Emoji 속성 문자만 이모지, 유니코드 기호(✦◆※❧⊕①②③⚡)는 제외
  - 리스트 항목(`<li>`) 내 이모지도 예산에 포함됨을 명시
  - cover/end 슬라이드는 이모지 예산에서 면제 (5개 이상은 자제)

### 3. 색상 대비 자동 조정 절차 추가 (P2)
- **문제**: 2/5 프레젠테이션에서 색상 대비 미달 (styles.md 팔레트 자체 문제)
- **개선**:
  - ⚠️ 마커 추가
  - 대비 미달 시 3단계 자동 조정 절차 명시:
    1. 밝은 배경 → 명도 10~15% 낮추기
    2. 어두운 배경 → 명도 10~15% 높이기
    3. 여전히 미달 → text/text-secondary 교환 또는 스펙 범위 내 대체 색상

### 4. CSS 변수 레거시 금지 규칙 추가 (P3)
- **문제**: 5/5 프레젠테이션이 --style-* 표준 변수와 --main, --white, --gray-* 레거시 변수 중복
- **개선**:
  - `--style-*` 표준 변수만 사용하도록 명시적 금지 규칙 추가
  - 허용 변수 목록: --style-bg, --style-primary, --style-accent, --style-text, --style-text-secondary + rgba() 변형

### 5. 색상 변수 동일값 예외 조건 추가 (P3)
- **문제**: Swiss International, Neo-Brutalism에서 --style-primary == --style-text 동일값
- **개선**:
  - 단색(monochrome) 팔레트 스타일에서 스펙이 동일 색상을 지정한 경우는 예외 허용

### 6. 카드 그리드 일관성 규칙 신설 (P2.5)
- **문제**: 2/5 프레젠테이션에서 card-grid 마지막 카드에 card-icon 누락
- **개선**:
  - 모든 카드에 일관되게 아이콘 포함하거나 모두 제외 규칙 추가
  - 이모지 예산 초과 시 색상 좌측 테두리(border-left)로 대체하는 방법 안내

### 7. 순차 등장 애니메이션 기본 적용 규칙 신설 (P2.5)
- **문제**: 3/5 프레젠테이션에서 staggered animation 미적용
- **개선**:
  - card-grid, stat-grid에 기본 staggered animation 적용 규칙 추가
  - CSS 코드 템플릿 포함 (popUp + nth-child delay)

### 8. 폰트 페어링 필수 규칙 강화 (핵심 디자인 원칙)
- **문제**: S3(Glassmorphism)에서 Noto Sans KR 단일 폰트만 사용
- **개선**:
  - ⚠️ 마커 추가, "단일 폰트 사용 금지" 명시
  - 최소 2개 폰트 패밀리(제목용 + 본문용) 필수
  - 한국어 지원 시 스펙 폰트 + 한국어 폰트 조합 가이드

### 9. 인라인 이벤트 핸들러 금지 규칙 신설 (P2)
- **문제**: S4(Swiss International) TOC에서 inline onclick/onkeydown 사용
- **개선**:
  - onclick, onkeydown, onmouseover 등 인라인 이벤트 핸들러 금지
  - `<script>` 블록에서 addEventListener로 등록하도록 명시

### 10. frameless 스타일별 예외 추가 (P2)
- **문제**: Neo-Brutalism에서 표지에도 프레임 유지가 스타일 정체성
- **개선**:
  - 두꺼운 테두리가 스타일 정체성인 경우 frameless 대신 프레임 유지 허용

## 기대 효과

- **section-icon 위반 제거**: 예외 목록 명시 + 경고 마커로 누락 방지
- **색상 대비 일관성**: 자동 조정 절차로 팔레트 자체 문제 해결
- **코드 품질 향상**: 레거시 CSS 변수 제거, 인라인 이벤트 금지로 유지보수성 개선
- **시각적 완성도**: staggered animation 기본 적용, 카드 아이콘 일관성, 폰트 페어링 필수화
- **규칙 모호성 해소**: 이모지 정의, 예산 면제, 동일값 예외 조건 명확화
